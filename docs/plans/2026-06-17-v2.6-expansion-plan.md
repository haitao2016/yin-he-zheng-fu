# V2.6 开发计划 — 舰种扩展 · 战斗深化 · 经济系统升级

> 创建时间: 2026-06-17
> 基线: V2.5（代码总量约 41,000 行）
> 主题: **新舰种 · 新技能 · Boss机制 · 新建筑 · 科技分支 · 新资源**
> 目标: 扩展战斗策略深度，丰富经济系统层次，增强Boss战斗体验

---

## 当前版本基线（V2.5）

| 维度 | 现状 |
|------|------|
| **舰种** | 8 种（SCOUT/FRIGATE/DESTROYER/BATTLECRUISER/ENGINEER/EXPLORER/CARRIER/INTERCEPTOR） |
| **战斗技能** | 6 个（全体集火/紧急修复/EMP冲击/护盾强化/相位加速/量子弹幕），Lv1-3升级 |

## A 系列：战斗系统扩展（高优先级）

### A1. 新舰种系统

- **问题**：8种舰船覆盖基础战斗角色，但缺乏特殊战术定位（隐形侦察、战场支援、超重型火力）；高端玩家追求更精细的舰队配置
- **方案**：新增 **3 种特殊舰船**，各有独特战术定位：

#### A1-1：隐形舰（STEALTH）

| 属性 | 数值 |
|------|------|
| 名称 | 隐形舰 |
| 速度 | 150 |
| 生命 | 60 |
| 攻击 | 8 |
| 射程 | 150 |
| 建造时间 | 25s |
| 建造费用 | 金属×200 + 能源×100 + 水晶×50 |
| **被动特性** | 战斗开始时隐身5s（不被攻击），隐身期间移动速度×2，现身后首次攻击伤害×1.5 |
| **战术定位** | 侦察骚扰、后排突袭、配合EMP打出爆发 |

#### A1-2：支援舰（SUPPORT）

| 属性 | 数值 |
|------|------|
| 名称 | 支援舰 |
| 速度 | 50 |
| 生命 | 200 |
| 攻击 | 0 |
| 射程 | 0 |
| 建造时间 | 30s |
| 建造费用 | 金属×300 + 能源×150 |
| **被动特性** | 每8s为半径120内血量最低友舰恢复15HP；相邻友舰攻击力+10%（光环叠加上限30%） |
| **战术定位** | 续战核心、保护旗舰、配合DESTROYER/BATTLECRUISER持久战 |

#### A1-3：巨型战舰（DREADNOUGHT）

| 属性 | 数值 |
|------|------|
| 名称 | 巨型战舰 |
| 速度 | 20 |
| 生命 | 5000 |
| 攻击 | 80 |
| 射程 | 450 |
| 建造时间 | 240s |
| 建造费用 | 金属×5000 + 能源×2000 + 核能×800 + 水晶×200 |
| **被动特性** | AOE半径100，每波次仅可建造1艘（上限锁定）；受击时10%概率触发"要塞反击"（对攻击者造成50伤害） |
| **战术定位** | 终局火力核心、Boss杀手、单舰顶替编队 |

- **解锁条件**：
  - STEALTH：基地核心Lv4 + 科技"相位驱动"解锁
  - SUPPORT：基地核心Lv3 + 科技"纳米修复"解锁
  - DREADNOUGHT：基地核心Lv8 + 科技"新星炮"解锁 + 巨构"量子堡垒"建成

- **文件修改**：
  - `GameConstants.lua`：新增 SHIP_TYPES 条目 + SHIP_COSTS + SHIP_QUEUE_ORDER
  - `BattleAI.lua`：新舰种AI行为 + 隐形/支援/反击逻辑
  - `FleetPanel.lua`：建造按钮 + 上限校验（DREADNOUGHT）
  - `TechPanel.lua`：科技解锁关联显示

- **规模**：中-大（约 200 行新增 + 80 行修改）

### A2. 新战斗技能系统

- **问题**：现有6个技能均为单体/全队增益，缺乏环境交互和群体协作机制；技能选择策略单一
- **方案**：新增 **3 种高级技能**，引入环境交互和群体协作：

#### A2-1：群体技能 — 战术协同（TACTICAL_SYNC）

| 属性 | 数值 |
|------|------|
| 名称 | 战术协同 |
| 冷却 | 60s |
| 持续 | 8s |
| 效果 | 激活期间，所有舰船攻击同一目标时伤害叠加（每艘+8%，上限50%）；击杀目标后全队获得2s加速×1.5 |
| 解锁 | 科技"虚空锚定"解锁 |
| 战术定位 | 集火Boss、快速清波、配合连锁反应 |

#### A2-2：环境技能 — 引力陷阱（GRAVITY_TRAP）

| 属性 | 数值 |
|------|------|
| 名称 | 引力陷阱 |
| 冷却 | 45s |
| 持续 | 6s |
| 效果 | 在指定区域（半径80）生成引力场，敌方舰船进入后移动速度×0.3且无法离开；场内敌舰每秒受10伤害 |
| 解锁 | 科技"曲速引擎"解锁 + 星域异象"引力井"首次遇到 |
| 战术定位 | 区域控制、分割战场、配合AOE舰船 |

#### A2-3：环境技能 — 暗能风暴（DARK_STORM）

| 属性 | 数值 |
|------|------|
| 名称 | 暗能风暴 |
| 冷却 | 90s |
| 持续 | 10s |
| 效果 | 全场进入暗能状态：敌舰攻击力×0.7、我方攻击力×1.2；但双方每秒均受5真实伤害 |
| 解锁 | 科技"新星炮"解锁 + 终局危机"暗能风暴"存活 |
| 战术定位 | 逆境翻盘、Boss决战、高风险高回报 |

- **技能栏扩展**：现有6技能 → 9技能，分两行显示（上排6基础 + 下排3高级）
- **技能点系统扩展**：高级技能升级消耗×2技能点

- **文件修改**：
  - `BattleSkills.lua`：新增技能状态 + CD + 效果函数
  - `BattleScene.lua`：技能触发逻辑 + 引力场渲染
  - `TechPanel.lua`：技能解锁关联
  - `GameConstants.lua`：技能解锁条件定义

- **规模**：中（约 150 行新增 + 60 行修改）

### A3. Boss机制扩展

- **问题**：Boss仅为数值放大的普通舰船，无特殊机制；击败Boss缺乏战术挑战感
- **方案**：Boss升级为 **多阶段 + 专属技能** 机制：

#### A3-1：多阶段Boss

| Boss类型 | 阶段数 | 各阶段特征 |
|----------|--------|------------|
| **战列Boss** | 3阶段 | 阶段1(HP>70%): 正常攻击<br>阶段2(HP30-70%): 攻击力×1.5 + AOE半径×2<br>阶段3(HP<30%): 攻击力×2 + 每5s召唤2艘护卫舰 |
| **母舰Boss** | 3阶段 | 阶段1(HP>60%): 正常攻击 + 无人机<br>阶段2(HP20-60%): 护盾值×3 + 无人机频率×2<br>阶段3(HP<20%): 自爆倒计时15s（需在倒计时内击杀否则全场AOE200伤害） |
| **虚空Boss**（新增） | 4阶段 | 阶段1: 隐形状态（仅被STEALTH/SCOUT可见）<br>阶段2: 现形 + 分裂为3个幻影（需击破真身）<br>阶段3: 真身HP<50% + 全场暗能腐蚀<br>阶段4: 真身HP<20% + 终局技能"虚空吞噬"（需集火打断） |

- **阶段转换动画**：Boss血条分段显示 + 阶段切换时全屏警告 + Boss短暂无敌1s

#### A3-2：Boss专属技能

| Boss技能 | 效果 | 触发条件 |
|----------|------|----------|
| **要塞炮击** | 对血量最高友舰造成200伤害 + 击退50px | 战列Boss阶段2，每10s |
| **无人机群** | 释放6架自爆无人机（每架HP20，接触友舰爆炸40伤害） | 母舰Boss阶段1-2，每8s |
| **虚空吞噬** | 全场友舰每秒受20伤害，持续5s；可被集火打断（累计伤害>500） | 虚空Boss阶段4，仅1次 |
| **幻影分身** | 生成2个幻影（HP=真身30%，攻击力=真身50%）；击破幻影不扣真身HP | 虚空Boss阶段2 |

- **Boss预警增强**：Boss波次前10s显示Boss类型 + 阶段预览 + 推荐配置

- **文件修改**：
  - `BattleAI.lua`：Boss阶段逻辑 + 专属技能触发
  - `BattleScene.lua`：Boss阶段渲染 + 血条分段 + 预警UI
  - `GameConstants.lua`：Boss阶段定义表
  - `MutantShipSystem.lua`：虚空Boss击杀奖励（变异舰船概率×2）

- **规模**：大（约 250 行新增 + 100 行修改）

## C 系列：经济与科技系统扩展（中优先级）

### C3. 新资源类型

- **问题**：3种原矿→3种精炼，资源链条单一；后期资源过剩缺乏稀缺性驱动
- **方案**：新增 **稀有矿物 + 能量晶体** 双层资源体系：

#### C3-1：稀有矿物（RARE_MINERALS）

| 类型 | 来源 | 用途 |
|------|------|------|
| **钛合金** | 高级精炼厂产出（概率5%，精炼速率×2时概率×2） | 巨型战舰建造必需×200；建筑专精激活×30 |
| **暗物质** | 虚空Boss击杀奖励×50；终局危机存活×100 | Tier5科技解锁必需；虚空舰建造×100 |
| **星核碎片** | 星域异象"暗物质涌流"事件×20 | 巨构工程阶段加速×10碎片/阶段 |

#### C3-2：能量晶体（ENERGY_CRYSTAL）

| 类型 | 来源 | 用途 |
|------|------|------|
| **蓝晶石** | 水晶小行星（新增类型）采矿产出 | 支援舰建造×50；科研站激活×30 |
| **紫晶石** | Boss击杀概率掉落×10 | 高级技能解锁×100；Boss预警强化×50 |
| **彩虹晶** | 终局危机"虚空蜂群"击杀×5 | 变异舰船词缀强化×20；指挥官经验×2 |

#### C3-3：小行星类型扩展

| 新类型 | 产出 | 出现概率 |
|--------|------|----------|

| **混合小行星** | 矿石×8 + 能量块×8 + 水晶×8 | 5%（高价值目标） |
| **虚空残骸** | 暗物质×5（仅终局危机期间出现） | 终局危机专属 |

- **资源栏扩展**：顶栏新增"稀有资源"折叠面板，显示6种稀有资源存量

- **文件修改**：
  - `GalaxyState.lua`：新增 ASTEROID_TYPES 条目

- **规模**：中-大（约 180 行新增 + 80 行修改）

## 技术风险评估

| 功能 | 风险 | 说明 |
