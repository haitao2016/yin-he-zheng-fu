# V3.0 银河征服·全面进化版 - 开发计划

> 创建时间: 2026-06-18
> 最后更新: 2026-06-18
> 状态: 待开始

---

## 1. 项目概述

**银河征服 V3.0** 在 V2.8 全部功能基础上，整合 V1.7/V1.8/V2.9 未实施内容，并加入 V3.0 级新特性，全量提升游戏内容广度、系统深度与体验品质。

**三大方向**：
- **内容缺口补全**：科技树 Tier4 路线、基地核心 Lv8-10、探索任务模板、无尽模式 Roguelike 化
- **广度与打磨**：星系事件扩充至 20+ 种、视觉动画打磨、UI/UX 精修、平衡性调优
- **V3.0 新特性**：多语言本地化、AI 难度分级系统、新舰种/新战役章节、存档系统增强

**技术栈**：UrhoX 引擎 + raw NanoVG 渲染 + 多人网络（Client/Server 架构）

## 3. Phase 1：核心内容缺口补全

### P0 级任务（最高优先级）

#### P0-1 科技树 Tier4 双路线补全

**涉及文件**：`scripts/game/Systems.lua`, `scripts/game/systems/ResearchSystem.lua`, `scripts/game/ui/TechPanel.lua`

**任务内容**：
- 补全 `ADVANCED_WEAPONS` 路线 Tier4：`NOVA_CANNON`（新星炮 - 范围型高能武器）
  - 解锁条件：Tier3 武器科技 + 基地 Lv8
  - 效果：主力舰武器伤害 +15%，范围攻击 +10%
- 补全 `DEFENSE_MATRIX` 路线 Tier4：`FORTRESS_PROTOCOL`（要塞协议 - 综合防御强化）
  - 解锁条件：Tier3 防御科技 + 基地 Lv8
  - 效果：全舰护甲 +20%，护盾恢复速率 +15%
- 加入对应研究点消耗与研究时长
- UI 面板新增 Tier4 节点显示

**验收标准**：科技树中两条路线 Tier4 可正常研究，加成在战斗中生效

#### P0-2 基地核心 Lv8-10 专属模块解锁

**涉及文件**：`scripts/game/systems/BaseBuildingSystem.lua`, `scripts/game/ui/BasePanel.lua`

**任务内容**：
- **Lv8 粒子加速器**：舰船建造速度 +25%，可解锁粒子武器模块
- **Lv9 主曲速门**：曲速航行速度 +30%，曲速能耗 -20%，解锁长距离跃迁
- **Lv10 恒星要塞**：全舰队生命值 +15%，解锁旗舰核心强化模块
- 每级解锁对应前置条件（资源量、前置建筑等级）
- 加入专属升级动画与解锁提示

**验收标准**：基地可从 Lv7 正常升级至 Lv10，每级加成生效

### P1 级任务

#### P1-1 探索任务模板扩展至 20+ 条（含链式任务）

**涉及文件**：`scripts/game/GalaxyEvents.lua`, `scripts/game/systems/GalaxyEventSystem.lua`

**任务内容**：
- 将现有事件模板从 ~14 条扩充至 20+ 条
- 新增事件类型维度：经济类（贸易风暴/资源发现）、战斗类（海盗围堵/雇佣兵招募）、探索类（远古遗迹/虫洞发现）、社交类（外交事件/异星种族接触）
- 引入链式事件机制：事件 A 触发 → 完成条件达成 → 解锁事件 B → 最终奖励
- 事件触发条件细化（舰队规模阈值、资源量、已研究科技、星系位置）

**验收标准**：新事件在游戏中可触发，链式事件能正确流转，所有事件 UI 显示正常

#### P1-2 无尽模式 Roguelike 化（轮间 3选1 选卡）

**涉及文件**：`scripts/game/systems/ChallengeSystem.lua`, `scripts/game/ui/EndlessPanel.lua`（如不存在则新建）

**任务内容**：
- 波次结算后进入"选卡阶段"，玩家从 3 张随机卡牌中选择 1 张
- 15 张卡池：
  - **增益类**（6张）：全舰攻击 +10%、全舰防御 +10%、护盾恢复 +20%、资源获取 +15%、建造速度 +20%、曲速 +15%
  - **减益但高回报类**（3张）：敌舰强度 +20% 但波次奖励 +50%、资源产出 -10% 但暴击率 +15%、舰队规模 -10% 但单体强度 +20%
  - **特殊效果类**（6张）：召唤援军舰队、获得随机稀有资源、解锁特殊舰种、全舰治疗 50%、获得指挥官技能、进入小行星带（资源战）
- 选卡动画与效果应用
- 选卡历史记录（本轮已选卡牌）

**验收标准**：每波结束后弹出选卡面板，卡牌效果在下一波战斗中生效

### P2 级任务

#### P2-1 战斗中途指令系统

**涉及文件**：`scripts/game/battle/BattleScene.lua`, `scripts/game/battle/BattleCombatPlayer.lua`, `scripts/game/ui/（战斗指令面板）`

**任务内容**：
- 战中可下达 3 类指令：
  - **集火目标**：全舰队集中攻击指定敌舰（需玩家点击选择）
  - **优先防御**：全舰转入防御姿态（伤害 -30%，防御 +30%，持续 10 秒）
  - **后撤重整**：舰队后撤并进行护盾紧急恢复（护盾 +30%，但 5 秒内无法攻击）
- 指令冷却时间与消耗
- UI 面板：战斗 HUD 中新增指令按钮区域

**验收标准**：战斗中可正常下达并执行指令，效果与冷却正确

#### P2-2 隐藏成就链式解锁（12+ 隐藏成就）

**涉及文件**：`scripts/game/systems/AchievementSystem.lua`, `scripts/game/ui/AchievementPanel.lua`

**任务内容**：
- 新增 12+ 隐藏成就，以 `???` 遮罩未解锁项
- 引入成就解锁链：
  - 链1（探索）：探索 5 个星系 → 探索 20 个星系 → 发现所有远古遗迹 → 发现隐藏星系
  - 链2（战斗）：无伤完成一波 → 无伤完成 5 波 → 无伤完成一整章战役
  - 链3（经济）：首次建造 10 艘舰 → 首次建造 50 艘舰 → 组建百舰舰队
  - 链4（社交）：完成首个公会任务 → 完成 10 个公会任务 → 成为公会领袖
- 隐藏成就解锁后显示真名与描述

**验收标准**：隐藏成就以 `???` 显示，解锁条件达成后正确解锁并显示名称

### P3 级任务

#### P3-1 快捷信号通信系统 + EndGame 战绩雷达图

**涉及文件**：`scripts/game/systems/FriendSystem.lua`, `scripts/game/ui/EndGamePanel.lua`

**任务内容**：

- 雷达图数据来源：战斗统计数据、经济数据、探索记录、指令使用记录

**验收标准**：信号在多人模式中正常显示，雷达图数据准确反映本局表现

#### P0-1 星系事件扩充至 20+ 种

**涉及文件**：`scripts/game/GalaxyEvents.lua`, `scripts/game/systems/GalaxyEventSystem.lua`

**任务内容**：
- 在 P1-1 基础上继续扩充至 25+ 种事件

  - **贸易风暴**：全星系市场价格剧烈波动 60 秒
  - **远古遗迹群**：连续触发遗迹探索事件链
  - **曲速异常**：曲速航行出现随机坐标偏移
  - **海盗集结**：特定星系出现大规模海盗舰队
  - **星门激活**：发现新的跃迁星门连接未知星系

**验收标准**：事件库 25+ 种，触发概率系统正常工作

**涉及文件**：`scripts/game/ui/` 目录各面板

**任务内容**：

- 信息层级优化：关键信息优先展示，次要信息可折叠
- 交互提示增强：按钮悬停提示、操作成功反馈、操作撤销确认

### P2 级任务

#### P2-1 平衡性全局调优

**涉及文件**：`scripts/game/GameConstants.lua`, `scripts/game/Systems.lua`

**任务内容**：
- **舰种再平衡**：按攻击/生命/造价三角重新审视所有舰种，确保每个舰种有明确定位
- **经济曲线调整**：前期资源获取更平滑，避免开局困境；后期增加资源消耗压力
- **科技树研究时间平衡**：确保各路线研究时长合理，无明显最优路径
- **事件奖励平衡**：事件奖励与风险匹配，无过度强势或过于弱势的事件

**验收标准**：通过数值审查确保平衡，无明显 OP/弱势选项

#### P2-2 随机星图可玩变体（种子系统 + 难度变体）

**涉及文件**：`scripts/game/GalaxyGenerator.lua`, `scripts/game/galaxy/GalaxyState.lua`

**任务内容**：
- **种子系统**：星图生成使用随机种子，玩家可输入/复现特定种子
- **难度变体**：开局选择星图变体
  - **标准**：默认资源分布与敌强度
  - **资源丰富**：资源点密度 +50%，敌强度 +10%
  - **资源贫瘠**：资源点密度 -30%，敌强度 -10%
  - **高威胁**：敌强度 +30%，但击败奖励 +50%
- 种子与变体信息在存档中记录

**验收标准**：相同种子生成相同星图，变体选项可正常选择并生效

## 5. Phase 3：V3.0 新特性突破

### P0 级任务

#### P0-1 AI 难度分级系统（4 档）

**涉及文件**：`scripts/game/battle/BattleAI.lua`, `scripts/game/GameConstants.lua`

**任务内容**：
- **简单**：敌舰生成 -30%，敌攻击 -20%，事件负面效果 -50%
- **普通**：默认参数（现有平衡值）
- **困难**：敌舰生成 +30%，敌攻击 +20%，资源获取 -10%
- **噩梦**：敌舰生成 +60%，敌攻击 +40%，资源获取 -20%，事件负面效果 +50%
- 难度选择在新游戏开始时提供
- 难度信息记录在存档中，影响排行榜显示（仅同等难度排行对比）
- AI 行为差异：简单 AI 攻击逻辑简化，噩梦 AI 增加战术变化（集火玩家脆弱舰、合理使用技能）

**验收标准**：新游戏可选择难度，4 档难度敌行为与数值差异明显

#### P0-2 多语言本地化实现（中文/英文双语）

**涉及文件**：`scripts/game/i18n/zh-CN.lua`, `scripts/game/i18n/en-US.lua`（新建）

**任务内容**：
- 在 P3-1 框架基础上，完成全量文本翻译
- en-US 文件：覆盖 UI 文本、事件描述、成就说明、科技树名称、舰种名称、系统提示等
- 翻译质量检查：专业术语一致性（如 "fleet" / "旗舰" / "曲速" 等）
- 运行时切换验证

**验收标准**：英文语言切换后所有文本正常显示，无未翻译残留

### P1 级任务

#### P1-1 新舰种扩展（2-3 种特色舰）

**涉及文件**：`scripts/game/GameConstants.lua`, `scripts/game/Systems.lua`, `scripts/game/systems/FleetManager.lua`, `scripts/game/ui/ShipDetailPanel.lua`

**任务内容**：
- **工程维修舰（ENGINEER）**：
  - 定位：支援舰，低攻击/高生存
  - 技能：周期性为周围友舰修复生命（范围治疗）
  - 造价：中等资源 + 需要工程科技解锁
- **隐形突击舰（STEALTH）**：
  - 定位：高爆发/低生存
  - 技能：周期性进入隐形状态（不可被选作目标，移动速度 +50%），攻击时解除隐身但造成额外伤害
  - 造价：高资源 + 需要隐身科技解锁
- **轨道炮舰（RAILGUN）**：
  - 定位：超远程单体高伤/慢速
  - 技能：充能后发射轨道炮，对单一目标造成巨额伤害（充能 3 秒，冷却 10 秒）
  - 造价：高资源 + 需要 Tier4 武器科技解锁

**验收标准**：新舰种可正常建造、参战，技能效果正确

- **奖励曲线调优**：赛季积分奖励曲线平滑化，确保各阶段有合理回报
- 赛季进度 UI 改进：当前积分/下一奖励进度条可视化

**验收标准**：主题赛季参数生效，限时赛季可配置

- 教学进度存档

**验收标准**：首次游戏流程中有清晰引导，可正常跳过
