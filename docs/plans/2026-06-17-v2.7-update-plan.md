**超级 Boss 定位**：
- 不同于普通 5 波/10 波 Boss，超级 Boss 在特定里程碑（波次 50/100/200）出现

### P0-2: 无尽波次模式

**模式入口**：主界面新增"无尽挑战"按钮

**无尽状态机**：
```lua
ENDLESS_MODES = {
    CLASSIC = { name = "经典无尽", startWave = 1, difficultyCurve = "linear" },
    SURVIVAL = { name = "生存模式", startWave = 20, difficultyCurve = "exponential" },
    SPEEDRUN = { name = "速通模式", startWave = 1, timeLimit = 600, goalWave = 50 },
}

-- 难度递增曲线
function getEndlessDifficulty(waveNum, mode)
    local baseMultiplier = 1.0
    if mode == "linear" then
        baseMultiplier = 1.0 + (waveNum - 1) * 0.08  -- 每波 +8%
    elseif mode == "exponential" then
        baseMultiplier = math.pow(1.1, waveNum - 1)  -- 每波 ×1.1
    end
    return math.min(10.0, baseMultiplier)  -- 上限 10x
end
```

**无尽奖励**：
- 每 10 波奖励：蓝晶石 30-50
- 每 25 波奖励：稀有材料
- 波次记录：最高波次保存

### P0-3: 每日挑战系统

**挑战生成**：
```lua
DAILY_CHALLENGES = {
    -- 单舰限制
    { type = "SHIP_LIMIT", shipType = "DESTROYER", count = 5 },
    { type = "NO_SUPPORT", desc = "禁止建造支援舰" },
    { type = "ONLY_STEALTH", desc = "只能建造隐形舰" },
    -- 资源限制
    { type = "LOW_RESOURCE", metalMult = 0.5, esourceMult = 0.5 },
    { type = "NO_MINES", desc = "禁止建造矿井" },
    -- 战斗限制
    { type = "NO_SKILLS", desc = "禁止使用主动技能" },
    { type = "BOSS_FROM_WAVE1", desc = "第1波就是Boss" },
    { type = "SPEED_BATTLE", timeLimitPerWave = 30 },
    -- 特殊条件
    { type = "HEALING_ONLY", desc = "只能建造支援舰和治疗舰" },
    { type = "MELEE_ONLY", desc = "只能建造近战舰" },
}

function generateDailyChallenge()
    local daySeed = os.date("%Y%m%d")  -- 每日固定种子
    math.randomseed(daySeed)
    -- 根据日期选择 1-2 个挑战组合
    local challenge = DAILY_CHALLENGES[math.random(#DAILY_CHALLENGES)]
    return challenge
end
```

**奖励**：
- 完成挑战：基础奖励 + 挑战积分
- 完美完成（无伤亡/无技能使用）：额外奖励

```lua
SHIP_ENHANCE_MAX = {
    DESTROYER = 20, BATTLECRUISER = 15, CARRIER = 10,
    STEALTH = 25, SUPPORT = 15, DREADNOUGHT = 8,
}
```

**强化逻辑**：
```lua
function enhanceShip(ship, material)
    local maxLv = SHIP_ENHANCE_MAX[ship.stype] or 10
    if ship.enhanceLevel >= maxLv then return false, "已达强化上限" end

    if not canAfford(material.cost) then return false, "资源不足" end
    spendResources(material.cost)

    ship.enhanceLevel = (ship.enhanceLevel or 0) + 1
    applyEnhancementEffects(ship, material.effect)

    return true, "强化成功！+" .. ship.enhanceLevel
end
```

### P0-5: 星际贸易路线

**贸易路线**：
```lua
TRADE_ROUTES = {
    -- 路线定义：{from, to, resource, amount, cooldown, profit}
    { from = "INDUSTRIAL_PLANET", to = "HOME_BASE", resource = "metal", amount = 50, cooldown = 60, profit = 1.2 },
    { from = "RESOURCE_RICH", to = "HOME_BASE", resource = "esource", amount = 30, cooldown = 90, profit = 1.3 },
    { from = "HOME_BASE", to = "FRONTIER", resource = "metal", amount = 40, cooldown = 120, profit = 1.5 },
}

-- 建立贸易路线条件
TRADE_ROUTE_REQUIREMENTS = {
    minDistance = 200,           -- 最小距离
    hasTradeHub = true,          -- 需要星际交易所
    fleetAvailable = true,        -- 有可用舰队
}
```

**贸易产出**：
- 每条路线按冷却时间自动产出资源
- 利润系数 > 1 表示资源产出 > 消耗
- 最多同时建立 3 条贸易路线

## 9. 技术风险与缓解

| 风险 | 等级 | 说明 | 缓解措施 |
|------|------|------|---------|
| 无尽模式数值溢出 | 高 | 波次 100+ 时伤害/血量可能超出数值范围 | 所有数值使用 math.min 硬上限（伤害 99999，血量 9999999） |
| 自动战斗 AI 决策质量差 | 中 | 简单 AI 可能导致玩家体验不佳 | 提供手动干预接口，玩家可随时接管 |
| 每日挑战种子可预测 | 低 | 玩家可能尝试逆向种子 | 使用日期 + 隐藏 salt 组合 |
| 超级 Boss 难度过高 | 中 | 新手无法通关导致流失 | 提供难度选择（普通/困难/噩梦） |
| 强化系统资源通胀 | 中 | 玩家通过刷材料快速强化破坏平衡 | 强化材料部分来自挑战奖励，不可交易 |
