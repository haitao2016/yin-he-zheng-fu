## V1.5 开发计划

### 当前版本基线（V1.4）

- **舰种**：8 种（SCOUT / FRIGATE / DESTROYER / BATTLECRUISER / ENGINEER / EXPLORER / CARRIER / INTERCEPTOR）
- **科技**：13 项，4 Tier，2 条研究路线
- **战斗技能**：6 个（集火 / 修复 / EMP / 护盾 / 加速 / 量子弹幕）
- **星图事件**：14 种随机事件
- **阶段目标**：15 个
- **成就**：15 个
- **难度**：简单 / 普通 / 困难（3 固定预设）
- **存档**：单槽（`galaxy_save.json`）
- **市场**：固定汇率（3 种精炼资源互换）

---

### P1 系列：核心体验升级（高优先级）

#### P1-1：战斗阵型系统
- **问题**：战斗开始后所有舰船直接以默认散乱阵型出场，无战前布阵环节，缺乏策略感
- **方案**：在战斗开始前 3 秒显示"阵型选择"提示，玩家可从 3 种阵型中选一：
  - **突击阵**（默认）：舰队集中于屏幕左中，全力冲锋
  - **防守阵**：舰队分散为两翼护卫，DESTROYER/BATTLECRUISER 前移护盾
  - **包围阵**：舰队上下分列，从两个方向夹击敌方
  - 阵型影响舰船初始 Y 坐标分布和前 5 秒的移动指令
- **文件**：`BattleScene.lua`（阵型 spawn 逻辑 + 阵型选择 UI）
- **规模**：中（约 160 行）

#### P1-2：自定义难度模式
- **问题**：现有简单/普通/困难三挡跨度较大，玩家无法微调到适合自己的节奏
- **方案**：难度选择界面新增第 4 项"自定义"，展开后开放 3 个滑块：
  - **海盗进攻间隔** 0.5× ~ 3.0×（越高越慢）
  - **初始资源** 0.5× ~ 2.0×（越高越多）
  - **海盗最高威胁等级** 1 ~ 7
  - 自定义难度不计入云端排行榜（显示"[自定义]"标签）
- **文件**：`Client.lua`（DIFFICULTY_CONFIGS 扩展 + 自定义配置序列化）、`GameUI.lua`（难度选择面板扩展）
- **规模**：中（约 110 行）

#### P1-3：任务日志面板
- **问题**：阶段目标散落在顶栏通知里，探索任务完成后无处回顾，玩家需靠记忆追踪进度
- **方案**：GameUI 银河界面右侧新增"日志"按钮，展开后显示两个标签页：
  - **目标**：15 个阶段目标按状态分组（✅已完成 / ⏳进行中 / 🔒未达成），进行中目标显示当前进度条
  - **探索记录**：最近 10 条探索任务完成记录（任务名 + 收益 + 时间戳）
- **文件**：`GameUI.lua`（新增日志面板渲染 + 按钮）、`Client.lua`（探索记录缓存）
- **规模**：中（约 140 行）

---

### P2 系列：玩法深度（中优先级）

#### P2-1：海盗进攻路径可视化
- **问题**：海盗舰队出发后直接出现在战场，星图上无任何预警提示，玩家难以提前迎击
- **方案**：海盗舰队进发时，在星图上从海盗基地到玩家基地绘制一条红色虚线攻击路径；路径上附带进攻倒计时标签（"⚠ 15s"）；当舰队进入基地防御范围时路径变为实线红色并闪烁
- **文件**：`GalaxyScene.lua`（进攻路径绘制函数）、`PirateAI.lua`（进攻状态暴露接口）
- **规模**：中（约 110 行）

#### P2-2：战斗中单舰信息面板
- **问题**：战斗中无法区分己方个别舰船的状态，集体技能触发后无从判断哪艘受益
- **方案**：点击战场中任意己方舰船，在画面右下角弹出该舰详细信息卡片：
  - 舰型图标 + 名称 + 当前血量/最大血量进度条
  - 本场击杀数 + 累计输出伤害
  - 当前攻击目标高亮红圈标记（连线绘制）
  - 点击空白处或再次点击取消选中
- **文件**：`BattleScene.lua`（选中状态 + 信息卡 UI + 攻击目标连线）
- **规模**：小（约 90 行）

#### P2-3：资源市场动态价格
- **问题**：市场汇率固定（1:1.5 等），玩家交易无时机判断，市场形同虚设
- **方案**：MarketSystem 引入价格波动机制：
  - 每 45 秒随机扰动各汇率 ±20%（不超出 0.6× ~ 1.8× 的合理范围）
  - 市场面板显示各交易对的当前价格 + 趋势箭头（▲/▼/→）
  - 价格历史迷你折线图（最近 5 个采样点）
  - 价格处于高位时显示"💰 有利"标签提示
- **文件**：`Systems.lua`（MarketSystem 波动逻辑）、`GameUI.lua`（市场面板 UI 扩展）
- **规模**：中（约 100 行）

---

### P3 系列：视觉润色（低优先级）

#### P3-1：成就徽章展示墙
- **问题**：成就面板是纯文字列表，缺乏收集感和视觉奖励
- **方案**：成就面板改为 3 列网格徽章布局：
  - 已解锁：彩色背景圆形徽章 + 图标 emoji + 名称，带金色外发光动画
  - 未解锁：灰色轮廓 + 问号图标，Hover 显示解锁条件提示
  - 顶部显示进度"已解锁 X / 15"及进度条
- **文件**：`GameUI.lua`（成就面板重构渲染）
- **规模**：小（约 100 行）

#### P3-2：战斗 MVP 统计
- **问题**：战斗结算和波次摘要仅有总体数据，不知道哪种舰型贡献最大
- **方案**：波次摘要弹窗下方新增"舰型贡献"一行，以横向迷你条形图展示本波各舰型击杀/伤害占比；战斗最终结算面板新增"MVP 舰种"高亮（贡献最高的舰型名称 + 金色勋章图标）
- **文件**：`BattleScene.lua`（各舰型击杀/伤害分别统计）、`GameUI.lua`（结算面板 MVP 区块）
- **规模**：小（约 90 行）

#### P3-3：星球殖民建议指数
- **问题**：PlanetPanel 展示基本信息，但玩家仍难判断"值不值得现在殖民这颗"
- **方案**：PlanetPanel 殖民按钮上方新增"战略评分"：
  - 综合 3 项指标：距基地距离（越近越高）、星球特产等级、竞争风险（附近海盗密度）
  - 1 ~ 5 星评分 + 主要优势文字标签（如"近距离 · 核能特产"）
  - 评分颜色：5★金 / 4★绿 / 3★白 / 1-2★灰
- **文件**：`PlanetPanel.lua`（评分渲染）、`Client.lua`（评分计算函数）
- **规模**：小（约 80 行）

### P1 系列：核心体验升级（高优先级）

#### P1-1：战斗时间线 / 波次进度条
- **问题**：玩家不知道当前是第几波、还差几波胜利，战斗节奏感弱，难以做出战术决策
- **方案**：战斗画面顶部中央新增波次进度条：
  - 横向轨道显示全部波次节点（圆点），已过波次实心，当前波次高亮脉冲，未来波次空心
  - Boss波次节点标红色骷髅图标
  - 进度条右侧显示"波次 3 / 8"文字 + 预计下波时间倒计时
  - 当前波次敌方舰队数量显示在进度条下方（"剩余 5 艘"）
- **文件**：`BattleScene.lua`（波次进度条渲染，约 80 行）
- **规模**：小

#### P1-2：战斗撤退 / 增援机制
- **问题**：战斗中无任何主动干预手段——不能逃跑也不能呼叫支援，感觉被动
- **方案**：战斗画面左下角新增两个战术按钮：
  - **撤退**（消耗 30 资源）：战斗判定为失败但基地 HP 仅损失 15%（而非正常损失），舰队保留 60% 存活舰船；每场战斗只能撤退一次
  - **紧急增援**（消耗 50 金属 + 50 晶体）：立即从储备池调入 2 艘 FRIGATE 加入当前战斗；仅在波次间隙（敌方清空后 3s 内）可用
- **文件**：`BattleScene.lua`（按钮渲染 + 撤退逻辑）、`Client.lua`（onBattleEnd 处理撤退结果）
- **规模**：中（约 120 行）

#### P1-3：星球建筑队列可视化
- **问题**：多星球同时建造时，玩家无法一眼看出哪颗星球在建什么、还要多久
- **方案**：银河界面每颗已殖民星球节点上方显示小型建造进度图标：
  - 如有建造队列：显示建筑图标（emoji）+ 环形进度弧（NanoVG arc）+ 剩余时间（秒）
  - 多个队列时叠加显示（最多 2 个，超出显示"+N"）
  - 建造完成时闪烁绿色提示并短暂放大
- **文件**：`GalaxyScene.lua`（星球节点扩展渲染，约 70 行）
- **规模**：小

### P2 系列：玩法深度（中优先级）

#### P2-1：舰队驻守星球系统
- **问题**：探索舰队（EXPLORER）只能执行探索任务，星球殖民后无驻守保护机制，缺乏防守层次
- **方案**：FleetPanel 中新增"驻守"指令：
  - 选择一支编队后可指定驻守目标星球
  - 驻守舰队不参与星图移动，但当该星球受到海盗侵扰（海盗特殊事件）时自动触发防御战
  - 驻守星球在星图上显示小型舰队图标（彩色小旗）
  - 驻守编队随时可召回变回机动编队
- **文件**：`FleetPanel.lua`（驻守指令 UI）、`Client.lua`（驻守状态逻辑 + 星球关联）、`GalaxyScene.lua`（驻守图标渲染）
- **规模**：中（约 150 行）

#### P2-2：战斗技能升级系统
- **问题**：6 个战斗技能固定强度，随着游戏进展没有成长感，后期感觉乏味
- **方案**：每完成 3 波战斗获得 1 个"技能点"，在波次间隙弹出"技能强化"选择（2选1）：
  - 每个技能有 3 级：Lv1（默认）→ Lv2（效果+50%）→ Lv3（效果+100%，冷却-20%）
  - 升级选择界面：两张卡片，显示技能名、当前等级、升级效果预览
  - 技能等级在战斗结算时清零（每局独立）
- **文件**：`BattleSkills.lua`（等级系统 + 效果缩放）、`BattleScene.lua`（技能升级弹窗 UI）
- **规模**：中（约 130 行）

#### P2-3：跨局成就奖励兑换
- **问题**：成就解锁后仅有通知和徽章，无实际游戏奖励，收集动力不足
- **方案**：成就面板新增"奖励"标签页：
  - 每个成就解锁后可"兑换"一次性初始加成（如：+100 金属起始 / 首波减少 1 艘敌舰 / 额外 1 个技能槽）
  - 兑换记录持久化到 `galaxy_career.json`，一个成就只能兑换一次
  - 已兑换的加成在下局游戏开始时自动生效，在难度选择界面显示"已激活 X 项成就加成"
- **文件**：`AchievementSystem.lua`（兑换逻辑）、`AchievementPanel.lua`（奖励 Tab UI）、`Client.lua`（开局应用加成）
- **规模**：中（约 110 行）

### P3 系列：视觉与体验润色（低优先级）

#### P3-1：战斗画面背景动态化
- **问题**：战斗背景是静态深空，与银河界面的粒子背景相比显得单调，缺乏沉浸感
- **方案**：为战斗场景添加动态背景层：
  - 缓慢飘移的星云雾气（半透明大圆渐变，NanoVG radialGradient，两层不同速度）
  - 偶发流星划过（1~3 条，随机方向，纯白渐变线条，寿命 1~2s）
  - 背景与当前战斗环境联动（星云环境加浓云雾，小行星带环境加漂移碎石）
- **文件**：`BattleScene.lua`（背景粒子层，约 90 行）
- **规模**：小

#### P3-2：星球面板动态数据图表
- **问题**：PlanetPanel 只显示当前时刻的资源产量，看不出历史趋势
- **方案**：PlanetPanel 底部新增迷你折线图区域（高 36px）：
  - 记录该星球最近 10 个时间点（每 30 秒采样）的资源产量
  - 3 条折线分别代表金属/晶体/能源（颜色与资源图标一致）
  - 折线图宽 200px，绘制于面板底部，显示"产量趋势"标题
- **文件**：`PlanetPanel.lua`（采样缓存 + 折线图渲染，约 80 行）、`Client.lua`（采样定时器）
- **规模**：小

#### P3-3：主界面沉浸式音乐切换
- **问题**：银河界面和战斗界面共用相同 BGM 风格，切换时无听觉节奏变化，战斗感不强
- **方案**：
  - 进入战斗场景时渐出银河 BGM（1.5s淡出），渐入战斗 BGM（1.5s淡入）
  - Boss 波次开始时将战斗 BGM 音调提升 5%（`AudioManager` pitch 控制）
  - 战斗胜利时播放短暂胜利音效，再渐回银河 BGM
  - Boss 击败时额外播放专属 fanfare（复用现有 `sounds/fanfare.ogg`）
- **文件**：`Client.lua`（场景切换音乐淡入淡出）、`AudioManager.lua`（pitch 控制 API）、`BattleScene.lua`（Boss波次音效触发回调）
- **规模**：小（约 60 行）
