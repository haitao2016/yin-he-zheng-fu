## 📁 项目结构

```
scripts/
├── main.lua                 # 入口
├── network/
│   ├── Client.lua           # 客户端主逻辑
│   ├── Server.lua           # 服务端逻辑
│   └── Shared.lua           # 共享常量/工具
└── game/
    ├── GalaxyScene.lua      # 银河地图渲染
    ├── BattleScene.lua      # 战斗场景
    ├── GameUI.lua           # UI 主模块（NanoVG）
    ├── PirateAI.lua         # 海盗 AI
    ├── Systems.lua          # 游戏系统（资源/科技/舰队）
    ├── AudioManager.lua     # 音频管理
    ├── AchievementSystem.lua# 成就系统
    └── ui/
        ├── UICommon.lua     # UI 公共工具
        ├── TopBar.lua       # 顶栏
        ├── BasePanel.lua    # 基地面板
        ├── FleetPanel.lua   # 舰队面板
        ├── TechPanel.lua    # 科技面板
        ├── PlanetPanel.lua  # 星球面板
        ├── NotifyPanel.lua  # 通知面板
        └── TutorialSystem.lua # 新手引导
```
