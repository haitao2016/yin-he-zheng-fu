## V1.5 开发计划

### 当前版本基线（V1.4）

- **舰种**：8 种（SCOUT / FRIGATE / DESTROYER / BATTLECRUISER / ENGINEER / EXPLORER / CARRIER / INTERCEPTOR）
- **科技**：13 项，4 Tier，2 条研究路线
- **战斗技能**：6 个（集火 / 修复 / EMP / 护盾 / 加速 / 量子弹幕）
- **星图事件**：14 种随机事件
- **阶段目标**：15 个
- **成就**：15 个
- **难度**：简单 / 普通 / 困难（3 固定预设）
- **存档**：单槽（`galaxy_save.json`）
- **市场**：固定汇率（3 种精炼资源互换）

---

### P1 系列：核心体验升级（高优先级）

#### P1-1：战斗阵型系统
- **问题**：战斗开始后所有舰船直接以默认散乱阵型出场，无战前布阵环节，缺乏策略感
- **方案**：在战斗开始前 3 秒显示"阵型选择"提示，玩家可从 3 种阵型中选一：
  - **突击阵**（默认）：舰队集中于屏幕左中，全力冲锋
  - **防守阵**：舰队分散为两翼护卫，DESTROYER/BATTLECRUISER 前移护盾
  - **包围阵**：舰队上下分列，从两个方向夹击敌方
  - 阵型影响舰船初始 Y 坐标分布和前 5 秒的移动指令
- **文件**：`BattleScene.lua`（阵型 spawn 逻辑 + 阵型选择 UI）
- **规模**：中（约 160 行）

#### P1-2：自定义难度模式
- **问题**：现有简单/普通/困难三挡跨度较大，玩家无法微调到适合自己的节奏
- **方案**：难度选择界面新增第 4 项"自定义"，展开后开放 3 个滑块：
  - **海盗进攻间隔** 0.5× ~ 3.0×（越高越慢）
  - **初始资源** 0.5× ~ 2.0×（越高越多）
  - **海盗最高威胁等级** 1 ~ 7
  - 自定义难度不计入云端排行榜（显示"[自定义]"标签）
- **文件**：`Client.lua`（DIFFICULTY_CONFIGS 扩展 + 自定义配置序列化）、`GameUI.lua`（难度选择面板扩展）
- **规模**：中（约 110 行）

#### P1-3：任务日志面板
- **问题**：阶段目标散落在顶栏通知里，探索任务完成后无处回顾，玩家需靠记忆追踪进度
- **方案**：GameUI 银河界面右侧新增"日志"按钮，展开后显示两个标签页：
  - **目标**：15 个阶段目标按状态分组（✅已完成 / ⏳进行中 / 🔒未达成），进行中目标显示当前进度条
  - **探索记录**：最近 10 条探索任务完成记录（任务名 + 收益 + 时间戳）
- **文件**：`GameUI.lua`（新增日志面板渲染 + 按钮）、`Client.lua`（探索记录缓存）
- **规模**：中（约 140 行）

---

### P2 系列：玩法深度（中优先级）

#### P2-1：海盗进攻路径可视化
- **问题**：海盗舰队出发后直接出现在战场，星图上无任何预警提示，玩家难以提前迎击
- **方案**：海盗舰队进发时，在星图上从海盗基地到玩家基地绘制一条红色虚线攻击路径；路径上附带进攻倒计时标签（"⚠ 15s"）；当舰队进入基地防御范围时路径变为实线红色并闪烁
- **文件**：`GalaxyScene.lua`（进攻路径绘制函数）、`PirateAI.lua`（进攻状态暴露接口）
- **规模**：中（约 110 行）

#### P2-2：战斗中单舰信息面板
- **问题**：战斗中无法区分己方个别舰船的状态，集体技能触发后无从判断哪艘受益
- **方案**：点击战场中任意己方舰船，在画面右下角弹出该舰详细信息卡片：
  - 舰型图标 + 名称 + 当前血量/最大血量进度条
  - 本场击杀数 + 累计输出伤害
  - 当前攻击目标高亮红圈标记（连线绘制）
  - 点击空白处或再次点击取消选中
- **文件**：`BattleScene.lua`（选中状态 + 信息卡 UI + 攻击目标连线）
- **规模**：小（约 90 行）

#### P2-3：资源市场动态价格
- **问题**：市场汇率固定（1:1.5 等），玩家交易无时机判断，市场形同虚设
- **方案**：MarketSystem 引入价格波动机制：
  - 每 45 秒随机扰动各汇率 ±20%（不超出 0.6× ~ 1.8× 的合理范围）
  - 市场面板显示各交易对的当前价格 + 趋势箭头（▲/▼/→）
  - 价格历史迷你折线图（最近 5 个采样点）
  - 价格处于高位时显示"💰 有利"标签提示
- **文件**：`Systems.lua`（MarketSystem 波动逻辑）、`GameUI.lua`（市场面板 UI 扩展）
- **规模**：中（约 100 行）

---

### P3 系列：视觉润色（低优先级）

#### P3-1：成就徽章展示墙
- **问题**：成就面板是纯文字列表，缺乏收集感和视觉奖励
- **方案**：成就面板改为 3 列网格徽章布局：
  - 已解锁：彩色背景圆形徽章 + 图标 emoji + 名称，带金色外发光动画
  - 未解锁：灰色轮廓 + 问号图标，Hover 显示解锁条件提示
  - 顶部显示进度"已解锁 X / 15"及进度条
- **文件**：`GameUI.lua`（成就面板重构渲染）
- **规模**：小（约 100 行）

#### P3-2：战斗 MVP 统计
- **问题**：战斗结算和波次摘要仅有总体数据，不知道哪种舰型贡献最大
- **方案**：波次摘要弹窗下方新增"舰型贡献"一行，以横向迷你条形图展示本波各舰型击杀/伤害占比；战斗最终结算面板新增"MVP 舰种"高亮（贡献最高的舰型名称 + 金色勋章图标）
- **文件**：`BattleScene.lua`（各舰型击杀/伤害分别统计）、`GameUI.lua`（结算面板 MVP 区块）
- **规模**：小（约 90 行）

#### P3-3：星球殖民建议指数
- **问题**：PlanetPanel 展示基本信息，但玩家仍难判断"值不值得现在殖民这颗"
- **方案**：PlanetPanel 殖民按钮上方新增"战略评分"：
  - 综合 3 项指标：距基地距离（越近越高）、星球特产等级、竞争风险（附近海盗密度）
  - 1 ~ 5 星评分 + 主要优势文字标签（如"近距离 · 核能特产"）
  - 评分颜色：5★金 / 4★绿 / 3★白 / 1-2★灰
- **文件**：`PlanetPanel.lua`（评分渲染）、`Client.lua`（评分计算函数）
- **规模**：小（约 80 行）

- **舰种**：8 种（SCOUT / FRIGATE / DESTROYER / BATTLECRUISER / ENGINEER / EXPLORER / CARRIER / INTERCEPTOR）
- **科技**：13 项 Tier1-3 + Lv8-10 专属模块 5 项
- **战斗技能**：6 个（集火 / 修复 / EMP / 护盾 / 加速 / 量子弹幕）
- **星图事件**：14 种随机事件，支持链式触发
- **阶段目标**：20 条（含链式事件目标）
- **成就**：15+ 个（含隐藏成就系统）
- **难度**：简单 / 普通 / 困难 / 无尽 / 自定义（5 种）
- **战斗特性**：阵型系统 · 战斗环境 · 连击奖励 · 集火指令 · 单舰信息 · DDA动态难度
- **UI 模块**：编队面板(左上角) · 任务日志 · 成就徽章 · 结算雷达图 · 局内统计面板

---

### P1 系列：核心体验升级（高优先级）

#### P1-1：战斗时间线 / 波次进度条
- **问题**：玩家不知道当前是第几波、还差几波胜利，战斗节奏感弱，难以做出战术决策
- **方案**：战斗画面顶部中央新增波次进度条：
  - 横向轨道显示全部波次节点（圆点），已过波次实心，当前波次高亮脉冲，未来波次空心
  - Boss波次节点标红色骷髅图标
  - 进度条右侧显示"波次 3 / 8"文字 + 预计下波时间倒计时
  - 当前波次敌方舰队数量显示在进度条下方（"剩余 5 艘"）
- **文件**：`BattleScene.lua`（波次进度条渲染，约 80 行）
- **规模**：小

#### P1-2：战斗撤退 / 增援机制
- **问题**：战斗中无任何主动干预手段——不能逃跑也不能呼叫支援，感觉被动
- **方案**：战斗画面左下角新增两个战术按钮：
  - **撤退**（消耗 30 资源）：战斗判定为失败但基地 HP 仅损失 15%（而非正常损失），舰队保留 60% 存活舰船；每场战斗只能撤退一次
  - **紧急增援**（消耗 50 金属 + 50 晶体）：立即从储备池调入 2 艘 FRIGATE 加入当前战斗；仅在波次间隙（敌方清空后 3s 内）可用
- **文件**：`BattleScene.lua`（按钮渲染 + 撤退逻辑）、`Client.lua`（onBattleEnd 处理撤退结果）
- **规模**：中（约 120 行）

#### P1-3：星球建筑队列可视化
- **问题**：多星球同时建造时，玩家无法一眼看出哪颗星球在建什么、还要多久
- **方案**：银河界面每颗已殖民星球节点上方显示小型建造进度图标：
  - 如有建造队列：显示建筑图标（emoji）+ 环形进度弧（NanoVG arc）+ 剩余时间（秒）
  - 多个队列时叠加显示（最多 2 个，超出显示"+N"）
  - 建造完成时闪烁绿色提示并短暂放大
- **文件**：`GalaxyScene.lua`（星球节点扩展渲染，约 70 行）
- **规模**：小

---

### P2 系列：玩法深度（中优先级）

#### P2-1：舰队驻守星球系统
- **问题**：探索舰队（EXPLORER）只能执行探索任务，星球殖民后无驻守保护机制，缺乏防守层次
- **方案**：FleetPanel 中新增"驻守"指令：
  - 选择一支编队后可指定驻守目标星球
  - 驻守舰队不参与星图移动，但当该星球受到海盗侵扰（海盗特殊事件）时自动触发防御战
  - 驻守星球在星图上显示小型舰队图标（彩色小旗）
  - 驻守编队随时可召回变回机动编队
- **文件**：`FleetPanel.lua`（驻守指令 UI）、`Client.lua`（驻守状态逻辑 + 星球关联）、`GalaxyScene.lua`（驻守图标渲染）
- **规模**：中（约 150 行）

#### P2-2：战斗技能升级系统
- **问题**：6 个战斗技能固定强度，随着游戏进展没有成长感，后期感觉乏味
- **方案**：每完成 3 波战斗获得 1 个"技能点"，在波次间隙弹出"技能强化"选择（2选1）：
  - 每个技能有 3 级：Lv1（默认）→ Lv2（效果+50%）→ Lv3（效果+100%，冷却-20%）
  - 升级选择界面：两张卡片，显示技能名、当前等级、升级效果预览
  - 技能等级在战斗结算时清零（每局独立）
- **文件**：`BattleSkills.lua`（等级系统 + 效果缩放）、`BattleScene.lua`（技能升级弹窗 UI）
- **规模**：中（约 130 行）

#### P2-3：跨局成就奖励兑换
- **问题**：成就解锁后仅有通知和徽章，无实际游戏奖励，收集动力不足
- **方案**：成就面板新增"奖励"标签页：
  - 每个成就解锁后可"兑换"一次性初始加成（如：+100 金属起始 / 首波减少 1 艘敌舰 / 额外 1 个技能槽）
  - 兑换记录持久化到 `galaxy_career.json`，一个成就只能兑换一次
  - 已兑换的加成在下局游戏开始时自动生效，在难度选择界面显示"已激活 X 项成就加成"
- **文件**：`AchievementSystem.lua`（兑换逻辑）、`AchievementPanel.lua`（奖励 Tab UI）、`Client.lua`（开局应用加成）
- **规模**：中（约 110 行）

### P3 系列：视觉与体验润色（低优先级）

#### P3-1：战斗画面背景动态化
- **问题**：战斗背景是静态深空，与银河界面的粒子背景相比显得单调，缺乏沉浸感
- **方案**：为战斗场景添加动态背景层：
  - 缓慢飘移的星云雾气（半透明大圆渐变，NanoVG radialGradient，两层不同速度）
  - 偶发流星划过（1~3 条，随机方向，纯白渐变线条，寿命 1~2s）
  - 背景与当前战斗环境联动（星云环境加浓云雾，小行星带环境加漂移碎石）
- **文件**：`BattleScene.lua`（背景粒子层，约 90 行）
- **规模**：小

#### P3-2：星球面板动态数据图表
- **问题**：PlanetPanel 只显示当前时刻的资源产量，看不出历史趋势
- **方案**：PlanetPanel 底部新增迷你折线图区域（高 36px）：
  - 记录该星球最近 10 个时间点（每 30 秒采样）的资源产量
  - 3 条折线分别代表金属/晶体/能源（颜色与资源图标一致）
  - 折线图宽 200px，绘制于面板底部，显示"产量趋势"标题
- **文件**：`PlanetPanel.lua`（采样缓存 + 折线图渲染，约 80 行）、`Client.lua`（采样定时器）
- **规模**：小

#### P3-3：主界面沉浸式音乐切换
- **问题**：银河界面和战斗界面共用相同 BGM 风格，切换时无听觉节奏变化，战斗感不强
- **方案**：
  - 进入战斗场景时渐出银河 BGM（1.5s淡出），渐入战斗 BGM（1.5s淡入）
  - Boss 波次开始时将战斗 BGM 音调提升 5%（`AudioManager` pitch 控制）
  - 战斗胜利时播放短暂胜利音效，再渐回银河 BGM
  - Boss 击败时额外播放专属 fanfare（复用现有 `sounds/fanfare.ogg`）
- **文件**：`Client.lua`（场景切换音乐淡入淡出）、`AudioManager.lua`（pitch 控制 API）、`BattleScene.lua`（Boss波次音效触发回调）
- **规模**：小（约 60 行）

## V2.0 开发计划

> 基线：V1.9（代码总量约 25,000 行）
> 目标：在现有玩法深度的基础上，新增**外交/同盟**维度、完善**星球个性化**体验、强化**无尽模式**可重玩性，同时修复已知体验痛点

---

### P1 系列：核心玩法扩展

#### P1-1：中立势力外交系统
- **问题**：目前银河地图只有玩家和海盗两方，缺乏政治博弈层次；中立星球只能殖民
- **方案**：
  - 新增 3 种中立势力标签（商业联盟 / 星际工会 / 遗迹守护者），随机标记于未殖民星球
  - 玩家可通过资源"外交礼物"提升关系（0–100 好感度），关系≥60 解锁商贸协议（每60s自动获取小额资源），关系≥90 解锁军事合作（海盗攻击该星球时自动协防）
  - 好感度低于0时势力宣战，攻击该星球（由现有PirateAI逻辑代理执行）
  - 外交操作入口：`PlanetPanel` 底部新增"外交"行（仅中立势力星球显示）
- **文件**：`Systems.lua`（势力数据+好感度逻辑）、`PlanetPanel.lua`（外交UI行）、`Client.lua`（事件触发+好感度存档）
- **规模**：中（约 160 行）

#### P1-2：动态星球事件——自然灾害
- **问题**：殖民后星球是静态的，缺少突发危机让玩家保持注意力
- **方案**：
  - 每180s对已殖民星球随机触发一次自然灾害（太阳风暴/小行星撞击/生态崩溃），概率3%/星球/次
  - 灾害使该星球产量降低50%，持续120s；玩家可消耗资源（能源×80）提前解除
  - 星图上受灾星球图标显示橙色警告光晕动画
  - 灾害触发挂接到 `GalaxyEvents` 系统（链式事件）
- **文件**：`GalaxyEvents.lua`（新增灾害事件类型）、`GalaxyScene.lua`（受灾星球渲染光晕）、`Systems.lua`（产量减益 debuff）
- **规模**：中（约 120 行）

#### P1-3：科技专精路线（Tech Specialization）
- **问题**：现有 16 项科技线性解锁，后期选择感不足，缺乏策略分叉
- **方案**：
  - Tier-3/4 科技各新增一个"专精替代科技"，玩家只能二选一
  - 例：Tier-3 `WARP_DRIVE` vs `VOID_ANCHOR`（速度+50% vs 进攻性+30%+防御+20%）
  - 选择后在 `TechPanel` 中另一路锁灰，并显示已选路线标签
  - 新增专精成就2条："路径坚守者"（同一专精路线赢得3场）、"全路线探索者"（两条路线各赢一场）
- **文件**：`Systems.lua`（新增4个专精科技定义+二选一逻辑）、`TechPanel.lua`（分叉渲染+锁灰效果）、`AchievementSystem.lua`（新增2条成就）
- **规模**：中（约 140 行）

### P2 系列：深度系统增强

#### P2-1：无尽模式改进——进阶选卡池与连胜加成
- **问题**：无尽模式选卡（ENDLESS_CARD_POOL）当前只有固定效果，10轮后提升感减弱
- **方案**：
  - 当前25张卡扩充至40张，新增15张史诗卡（橙色边框），每5轮起始时必出1张史诗卡
  - 新增"连胜狂潮"机制：连续3轮≥80%全消消灭率时触发，下一轮资源收入×1.5持续1轮
  - 选卡界面史诗卡添加金色粒子特效（nvgLinearGradient 实现）
  - 新增无尽模式独立排行榜（cloudScore `endless_score` 键）
- **文件**：`Client.lua`（卡池扩充+连胜判断+排行榜写入）、`BattleScene.lua`（连胜UI提示）、`GameUI.lua`（无尽排行榜展示）
- **规模**：中-大（约 200 行）

#### P2-2：战斗地图多区域布局
- **问题**：战斗场景全局固定视口，敌人总从右方直线推进，策略单一
- **方案**：
  - 保留现有战斗逻辑不变，在 `BattleScene` 视口新增"上下夹击"模式：随机波次中25%概率敌人从右侧上下两个入口同时进入
  - 两入口各生成50%的本波兵力，要求玩家在垂直方向合理分布舰队
  - 入口处显示橙色三角箭头预警动画（提前5s闪烁）
  - 新增相关成就："两线作战"（成功防守一次夹击波次）
- **文件**：`BattleScene.lua`（夹击波次生成逻辑+双入口渲染）、`AchievementSystem.lua`（新增成就）
- **规模**：中（约 130 行）

#### P2-3：星球建筑专精槽
- **问题**：建筑建造后效果固定，缺少后期自定义空间
- **方案**：
  - 每个已升3级的建筑新增一个"专精槽"，玩家可消耗晶石×50为其附加一个被动专精（共6种轮换）
  - 专精示例：矿场专精"深层钻探"（+15%矿石）、能源厂专精"热核循环"（+20%能源，-5%核能）
  - `PlanetPanel` 建筑行右侧新增小圆形专精槽图标，点击弹出选择模态框
  - 专精状态写入星球存档
- **文件**：`PlanetPanel.lua`（专精槽UI）、`Systems.lua`（专精数据+建筑效果计算更新）、`Client.lua`（专精存档读写）
- **规模**：中（约 150 行）

### P3 系列：体验打磨与视觉提升

#### P3-1：排行榜个人主页弹窗
- **问题**：排行榜仅显示分数列表，点击他人名字无任何额外信息，缺乏社交感
- **方案**：
  - 排行榜列表项点击后弹出"玩家名片"面板（280×200px）
  - 展示：头像占位圆 + 昵称 + 分数 + 殖民数 + 击杀数 + 成就解锁率（已解锁/总数%）
  - 面板通过 `clientCloud:GetUserInfo()` 拉取数据，500ms内加载
  - 动画：从点击位置缩放弹出（scale 0→1，0.2s）
- **文件**：`GameUI.lua`（名片弹窗渲染+点击路由）
- **规模**：小-中（约 80 行）

#### P3-2：战斗场景舰船血条动态
- **问题**：舰船血条是静态填充色块，受击时仅闪白，低血时无视觉紧张感
- **方案**：
  - 血量 < 30% 时血条由绿→红，并每0.5s闪烁（alpha 200↔255 lerp）
  - 受击瞬间血条显示 `-N` 飘字（白色，1.5s上浮淡出）

### P1 系列：核心体验升级（高优先级）

#### P1-1：舰船被动专属能力

- **问题**：8 种舰型仅有属性数值差异（HP/ATK/SPD），缺乏专属能力，组队策略感薄弱
- **方案**：为每种舰型增加一个**被动特性**，战斗中自动触发：
  - **SCOUT**：先敌洞察——进入战场时使当轮敌方全体攻击力 -10%（显示"侦察压制"浮字）
  - **FRIGATE**：协同护卫——血量>50% 时与身旁最近友舰形成"护卫对"，受击时互相分担 20% 伤害
  - **DESTROYER**：穿甲弹——每 5 次攻击触发一次穿甲（伤害×2，无视护盾层）
  - **BATTLECRUISER**：重甲要塞——受击时 10% 概率完全格挡（0 伤害），触发时显示"格挡"浮字
  - **ENGINEER**：战场维修——每 30s 为血量最低的友舰恢复 10 HP
  - **EXPLORER**：图绘先行——战斗开始标记敌方 HP 最高目标（红圈），友舰优先集火
  - **CARRIER**：舰载机群——每波次前额外生成 2 架临时战斗机（高 ATK 低 HP，存活 5s）
  - **INTERCEPTOR**：超音速穿越——攻速比其他舰型快 30%，击杀后 20% 概率立即再攻击一次
- **文件**：`BattleScene.lua`（被动触发逻辑 + 浮字/高亮渲染）、`Shared.lua`（被动属性定义表）
- **规模**：中-大（约 220 行）

#### P1-2：星球升级系统

- **问题**：殖民后星球一直保持初始状态，缺乏长期投入感和视觉成长
- **方案**：每颗已殖民星球新增**星球等级**（1-5 级），消耗资源升级：
  - Lv1→2（金属×200）/ Lv2→3（金属×500+晶体×200）/ Lv3→4（三资源×500）/ Lv4→5（三资源×1000+完整建筑组）
  - 每升一级：建筑槽上限 +1（从 4→最多 8 个）、全建筑产量 +5%（乘算）、星球光晕颜色变化
  - PlanetPanel 新增"升级星球"按钮，显示当前等级和升级所需资源对比
  - 升级动画：星球图标旋转光圈 + 等级数字浮现
- **文件**：`PlanetPanel.lua`（升级 UI+进度渲染）、`Systems.lua`（BuildingSystem 槽位扩展+产量计算）、`GalaxyScene.lua`（等级视觉差异）、`Client.lua`（升级存档读写）
- **规模**：中（约 160 行）

#### P1-3：战斗连锁反应系统

- **问题**：战斗全靠总伤害堆叠，缺乏"爆发时刻"的快感与策略操控感
- **方案**：新增**连锁爆炸**机制，同帧击杀 3 艘及以上敌舰时触发：
  - 被击杀舰周围半径内的敌舰额外受到 20% AOE 伤害（向外扩散一次，不递归）
  - 屏幕轻微震动（0.3s）+ "CHAIN ×N" 大字飘字 + 橙色冲击波圆圈
  - 连锁触发次数计入战斗结算统计页
  - 新增成就：**"连锁大师"**（单次战斗触发 3 次以上连锁）
- **文件**：`BattleScene.lua`（连锁判定 + 爆炸渲染 + 统计记录）、`AchievementSystem.lua`（新成就）
- **规模**：中（约 100 行）

### P1 系列：核心留存优化（高优先级）

#### P1-1：星际传承进化系统

- **问题**：每局相互独立，career 数据仅供展示，对下一局无实质影响；玩家精通后重玩动力大幅下降
- **方案**：新增跨局永久进化层"**星际文明树**"：
  - 每赢一局获得"文明积分"（普通=1、困难=2、无尽每层=0.5），积分存入 `galaxy_career.json`
  - 文明树分 3 条路线（军事/经济/科研），每条 4 个节点，共 12 项永久被动加成
  - **军事线**：Lv1 久战之师（首波攻击+8%）→ Lv2 钢铁洪流（DESTROYER/BATTLECRUISER HP+10%）→ Lv3 连击共鸣（连锁触发阈值由3降至2）→ Lv4 精英旗舰（CARRIER 建造时间-20%）
  - **经济线**：Lv1 勤劳星民（初始金属+200）→ Lv2 高效精炼（精炼速率+15%）→ Lv3 市场老手（市场交易折扣+10%）→ Lv4 帝国基建（首个建筑免费）
  - **科研线**：Lv1 求知欲（首项科技研究速度+25%）→ Lv2 技术储备（初始解锁一项 Tier1 科技）→ Lv3 分叉洞见（专精科技两路均可解锁一次）→ Lv4 量子突破（Tier4 科技费用-30%）
  - 每个节点解锁后立刻永久生效，在难度选择界面显示"已激活 N 项传承加成"
  - 文明树 UI 新增主菜单"**传承**"按钮（银河图标旁），面板含进度条和解锁动画
- **文件**：`Client.lua`（文明积分计算+传承加成应用）、`GameUI.lua`（传承树面板渲染）、`AchievementSystem.lua`（新增2条传承成就）
- **规模**：中-大（约 200 行）

#### P1-2：主动远征系统

- **问题**：游戏全程被动防守——等海盗来袭、等资源积累、等科技研究；玩家缺乏主动打出关键手的时机
- **方案**：FleetPanel 新增"**远征**"指令：
  - 选择一支编队（最少含 1 艘 FRIGATE 以上舰船）后，可指定任意已探明的海盗基地发动远征
  - 远征自动结算（无战斗 UI），耗时 = 距离 / 舰队平均速度（20~60s），期间编队不可用
  - 结算公式：击破率 = 舰队战力评分 / 海盗基地威胁评分，击破率决定战利品和伤亡
    - 大胜（>75%）：获得金属/能源/核能各 200~400，舰队损失 10~20%，海盗基地延迟进攻 30s
    - 小胜（40~75%）：获得资源各 80~150，舰队损失 30~40%
    - 失败（<40%）：舰队损失 50~70%，无战利品，但海盗下次进攻规模减小 15%
  - 星图上显示远征路径动画（橙色虚线，舰队小图标沿路径移动）
  - 战略决策：主力舰队用于远征 vs 留守防御，是游戏核心取舍
- **文件**：`FleetPanel.lua`（远征按钮+目标选择）、`Client.lua`（远征计时+结算逻辑）、`GalaxyScene.lua`（远征路径动画渲染）
- **规模**：中-大（约 180 行）

#### P1-3：帝国运营总览面板

- **问题**：殖民 5+ 颗星球后，玩家需频繁切换 PlanetPanel 才能了解各星球状态，多建筑队列管理混乱
- **方案**：银河界面新增"**帝国总览**"（E 键/顶栏按钮）全屏面板：
  - 网格布局，每颗已殖民星球一张卡片，显示：星球名、资源产量（金属/能源/核能/s）、建筑数/槽位、当前建造队列（最多2项）及剩余时间
  - 资源产量用彩色数字+趋势箭头（↑/↓）
  - 点击任意卡片直接跳转对应 PlanetPanel
  - **批量建造**：勾选多颗星球后，点击"同步建造"可为所有勾选星球加入相同建筑到队列（资源充足时直接建，不足时排队等待）
  - 底部显示帝国整体产量汇总（所有星球之和）
- **文件**：`GameUI.lua`（总览面板渲染函数，约120行）、`Client.lua`（批量操作逻辑）
- **规模**：中（约 150 行）

### P2 系列：玩法深度扩展（中优先级）

#### P2-1：每日星际挑战模式

- **问题**：主线模式（普通/困难/无尽）已充分探索后，缺乏新鲜的短期目标驱动每日回访
- **方案**：主菜单新增"**每日挑战**"模式按钮（带倒计时显示"23:45 后刷新"）：
  - 挑战配置由客户端按日期种子生成（不依赖网络），保证每天不同
  - 每日挑战包含 1个限制 + 1个强化：
    - 限制示例：仅可建造 SCOUT/FRIGATE/INTERCEPTOR（禁用大型舰）、星球建筑槽上限-1、科技研究速度×0.5
    - 强化示例：初始科技随机解锁一项 Tier2、首波延迟+30s、市场汇率固定为最优
  - 难度固定为"普通"（保证公平性），完成后上传专属排行榜（`challenge_YYYYMMDD` 键）
  - 连续完成 3/7/15/30 天解锁专属成就，第 7 天颁发"星际挑战者"限定头衔称号（显示在排行榜名称前）
  - 挑战限制在游戏开始时用横幅明确提示
- **文件**：`Client.lua`（挑战配置生成+规则验证）、`GameUI.lua`（挑战模式横幅+专属排行榜标签）、`AchievementSystem.lua`（连续挑战成就系列）
- **规模**：中（约 130 行）

#### P2-2：战役模式——银河征服序章

- **问题**：当前仅有"随机局"模式，缺乏叙事和剧情驱动的目标感；新玩家进入后不知道为什么而战
- **方案**：新增"**战役序章**"（固定剧情的 3 关教学战役）：
  - **第1关：拓荒**：固定小地图（4颗星球），唯一海盗基地，4波，着重教学殖民/建筑/造舰
    - 剧情文字框（3行）："星际联邦委派你统帅第七舰队，出发前往未知星域……"
  - **第2关：反击**：固定地图，2个海盗基地，8波含1次Boss，教学科技/技能
  - **第3关：联盟**：引入外交系统，必须与中立势力达成协议才能在10波内守住基地
  - 每关完成后解锁专属台词总结页 + 给予"文明积分" × 3（新玩家跳票传承进化）
  - 战役入口在主菜单中央，初次进入时自动引导
- **文件**：`Client.lua`（战役关卡配置 + 固定地图生成 + 剧情台词触发）、`GameUI.lua`（剧情文字框渲染）
- **规模**：中-大（约 200 行）

#### P2-3：波次间战术备战期

- **问题**：波次切换几乎无过渡，玩家处于持续反应状态，无法进行短暂的战术思考；波次间的短暂间隔没有被充分利用
- **方案**：每波清空后新增 **8 秒备战期**（原 3 秒倒计时扩展）：
  - 屏幕顶部出现"备战中 7s… 🔧"倒计时条（玩家可按 SPACE 跳过）
  - 备战期内战斗画面右下角展示**下波敌情预报**：显示下波舰型组成（图标+数量），Boss 波显示红色骷髅警告
  - 同时显示己方舰队简明状态：每艘存活舰船的 HP 百分比（以小格子图示，不遮挡战场）
  - 备战期结束后正常倒计时，无缝衔接下波
  - 关键效果：玩家在此期间可在星图执行建造指令（切换到星图再切回），让战斗和运营节奏更协调
- **文件**：`BattleScene.lua`（备战期状态 + 下波预报 UI + 跳过逻辑，约 90 行）
- **规模**：小-中（约 90 行）

### P3 系列：体验打磨（低优先级）

#### P3-1：战场伤害数字精细化

- **问题**：所有伤害数字大小一致、颜色单一，视觉上难以区分普通打击和关键打击，战斗爽感不足
- **方案**：伤害飘字按量级差异化展示：
  - **小伤害**（<20）：小字号（14px）白色，快速淡出（0.8s）
  - **中伤害**（20-100）：中字号（18px）黄色，正常淡出（1.2s）
  - **大伤害**（100+）：大字号（24px）橙色加粗，带轻微缩放动画（1.5s）
  - **连锁爆炸**：沿用现有橙色大字，字号升为 28px，带描边
  - **护盾吸收**：蓝色飘字（如"🛡-45"），图标与数字组合
  - **格挡/免疫**：灰色"格挡"文字，区别于伤害数字
  - **回复**：绿色"+"飘字（ENGINEER 修复/波次修复）
- **文件**：`BattleScene.lua`（伤害飘字渲染分级逻辑扩展，约 60 行）
- **规模**：小（约 60 行）

#### P3-2：科技树可视化 2.0

- **问题**：TechPanel 当前是带竖线的列表布局，Tier 间依赖关系不直观，专精分叉（exclusiveGroup）在视觉上表现不清晰
- **方案**：重构 TechPanel 为**横向树状布局**：
  - Tier1-4 横向排列，每 Tier 为一列（列宽自适应）
  - 科技节点之间用有向箭头连线（prereqs → 当前），连线颜色：绿色（已解锁前置）、灰色（未解锁前置）、红色（专精互斥锁）
  - 同一 exclusiveGroup 的互斥科技用虚线红框框住，顶部标注"二选一"
  - 已解锁科技绿色填充，研究中科技带进度条填充，可研究科技蓝色高亮脉冲
  - Hover 任意节点展示：效果说明、前置科技、互斥科技、解锁后的下游科技链
  - 面板底部显示推荐研究路线（根据当前战局状态自动高亮1条最优路径）
- **文件**：`TechPanel.lua`（布局算法重构+连线绘制+Hover详情面板，约 130 行）
- **规模**：中（约 130 行）

#### P3-3：音频体验优化

- **问题**：当前音量仅有主音量一挡，无法分别控制音效/BGM；部分音效（爆炸连续触发时）会堆积导致音频破音；战斗高潮时缺乏音乐强度变化
- **方案**：
  - **分离音量控制**：设置面板新增 BGM 音量 / 音效音量独立滑块（替代单一主音量），写入存档
  - **音效限声**：同类音效（如爆炸）设置最大并发数=3，超出时停止最老的一条，防止堆积破音
  - **自适应音乐强度**：波次开始后如当前存活敌舰数>当前波次期望值的 130%（形势危急），自动将战斗 BGM pitch 提升 5%（已有 pitch 控制 API）；危机解除后恢复
  - **胜负音效升级**：胜利时增加短暂 2 秒静音过渡再播放 fanfare（制造戏剧感）；失败时新增低沉"失败弦乐"音效（需要生成资源）
- **文件**：`SettingsPanel.lua`（新增双音量滑块）、`Client.lua`（分离音量写入存档）、`AudioManager.lua`（限声机制+自适应 pitch 逻辑）、`BattleScene.lua`（危机判断触发 pitch 变化）
- **规模**：小-中（约 80 行）

### P1 系列：核心玩法拓展（高优先级）

#### P1-1：舰船改装系统（Module Loadout）

- **问题**：8 种舰型虽各有被动特性，但同型舰完全相同——100 艘 DESTROYER 和 1 艘没有区别；玩家缺乏对单舰个性化养成的投入感
- **方案**：新增**改装模块**系统，每艘舰可装备 1 个模块（消耗"残骸零件"——战斗胜利掉落）：
  - **攻击模块**（5 种）：
    - 穿透弹头（攻击穿透 1 个目标，对后排 50% 伤害）
    - 速射装置（攻速+20%，伤害-10%）
    - 等离子灼烧（攻击附加 3s DOT，总计额外 15% 伤害）
    - 精准打击（暴击率 15%，暴击伤害 ×1.8）
    - 脉冲过载（每第 3 次攻击伤害 ×2.5，但需要 1s 充能间隔）
  - **防御模块**（5 种）：
    - 纳米装甲（HP+25%）
    - 能量护盾（战斗开始获得等同 30% HP 的护盾，不可恢复）
    - 应急维修（HP 低于 20% 时触发一次 50% 生命恢复，每场限 1 次）
    - 反射镀层（受击时 8% 概率将伤害反弹给攻击者）
    - 隐匿涂装（不会被优先集火，敌方随机选择目标）
  - **辅助模块**（5 种）：
    - 战术链路（相邻友舰攻击力+8%）
    - 侦察探针（战场开始时揭示所有敌舰属性面板 5s）
    - 牵引光束（击杀敌舰时 30% 概率获得额外残骸零件）
    - 超频引擎（移动速度+40%，用于远征时间减少）
    - 量子锚定（被击杀时 20% 概率复活，HP=1，持续 3s 无敌）
  - 改装 UI：FleetPanel 内舰船卡片下方新增"⚙ 改装"按钮，弹出模块选择面板
  - 残骸零件：每场战斗胜利获得 2-5 个（无尽模式每层+1），改装消耗 5 个/模块
  - 模块可卸下（免费）、替换（消耗 3 残骸），已卸下模块归入仓库
- **文件**：`Client.lua`（模块存档+属性注入战斗）、`FleetPanel.lua`（改装 UI 面板）、`BattleScene.lua`（模块效果触发逻辑）、`Shared.lua`（模块定义表）
- **规模**：大（约 280 行）

#### P1-2：宿敌系统（Nemesis Pirates）

- **问题**：海盗始终是匿名数字单位，击败后无任何记忆点；缺乏"私人恩怨"式的叙事驱动力
- **方案**：新增 **3 位命名宿敌海盗船长**，贯穿整个游戏生涯：
  - **生成规则**：首次被海盗击败后（或首次击败海盗 Boss 后），系统生成 1 位宿敌船长
    - 船长属性从 3 个模板随机：{"铁爪"卡洛斯（重火力型）、"幽灵"莉娜（隐身偷袭型）、"疯博士"奥兹（科技干扰型）}
    - 每位宿敌记忆玩家上次的**获胜策略**（主力舰型 + 使用技能），下次出现时针对性调整
  - **进化机制**：
    - 玩家每胜宿敌 1 次 → 宿敌下次 HP+15%、ATK+10%、新增 1 个战术（最多 3 个）
    - 宿敌每胜玩家 1 次 → 宿敌嘲讽台词+掠夺玩家 10% 当前金属（存档记录）
    - 宿敌达到 3 级后触发"**最终决战**"事件（星图专属红色标记），击败后永久消灭并获得专属传说模块
  - **战术列表**（每级随机获得 1 个）：
    - 火力压制：首波攻击力 ×1.5
    - 伏击阵型：从两侧夹击而非正面
    - 干扰力场：玩家技能冷却时间 +50%
    - 自爆无人机：被击杀时释放 AOE（半径内友舰受 30 伤害）
    - 量子闪避：20% 概率闪避攻击
  - **UI 表现**：宿敌出场时专属入场动画（屏幕暗化 + 船长肖像 + 嘲讽台词 1s）；击败时"宿敌陨落"大字 + 专属奖励弹窗
  - **新增成就**：
    - "恩怨终结"（击败首个满级宿敌）
    - "赏金猎人"（击败全部 3 位满级宿敌）
- **文件**：`Client.lua`（宿敌生成+进化+存档）、`BattleScene.lua`（宿敌战术执行+入场动画）、`GalaxyScene.lua`（宿敌标记渲染）、`AchievementSystem.lua`（新成就）
- **规模**：大（约 300 行）

#### P1-3：星际联赛模式（Weekly League）

- **问题**：每日挑战解决了日活问题，但缺乏中长期竞争目标；玩家之间无法在"相同条件"下比拼策略深度
- **方案**：新增"**星际联赛**"——每周一个固定种子的完整局，所有玩家在相同初始条件下竞速：
  - **规则**：固定星图种子 + 固定海盗刷新序列 + 固定市场波动，玩家自由决策经营和战斗
  - **计分**：综合评分 = 存活波次×10 + 殖民星球数×50 + 击杀总数×2 + 剩余资源÷100 + 效率时间奖励
  - **时间奖励**：游戏内用时越短（相同波次下），分数越高（鼓励高效决策）
  - **排行榜**：专属 `league_YYYYWW`（年+周号）排行榜，每周日 24:00 结算
  - **段位系统**（本地计算，不存服务端）：
    - 前 5%：传奇指挥官（金色边框）
    - 5-20%：精英舰长（紫色边框）
    - 20-50%：资深船长（蓝色边框）
    - 50%+：新锐航海家（绿色边框）
  - **入口**：主菜单"联赛"按钮（奖杯图标），内含本周排行榜 + 个人最佳 + 段位展示
  - **限制**：每周只能提交 1 次成绩（防刷分），提交前确认弹窗
  - **新增成就**："联赛之星"（首次进入前 20%）
- **文件**：`Client.lua`（联赛种子生成+计分公式+提交逻辑）、`GameUI.lua`（联赛面板+段位渲染）、`AchievementSystem.lua`（新成就）
- **规模**：中-大（约 180 行）

### P2 系列：玩法深度扩展（中优先级）

#### P2-1：星域异象系统（Sector Anomalies）

- **问题**：所有星球战斗环境完全相同，星球位置仅影响距离而不影响战斗本身；战术决策只发生在"造什么舰"层面
- **方案**：每颗星球生成时随机附加 0-1 个**星域异象**，影响在该星球发生的所有战斗：
  - **引力井**（GRAVITY_WELL）：所有舰船移动速度-30%，重型舰（BATTLECRUISER/CARRIER）不受影响
  - **离子风暴**（ION_STORM）：每 10s 全场随机 1 艘舰船受到 15 点真实伤害（无视护盾），概率对己方/敌方均等
  - **星云遮蔽**（NEBULA_CLOAK）：战场能见度降低，舰船只攻击半径 60% 以内的目标（缩短射程）
  - **太阳耀斑**（SOLAR_FLARE）：每 20s 触发一次全场 AOE 8 伤害，ENGINEER 的维修频率加倍生效
  - **暗物质涌流**（DARK_MATTER）：所有舰船攻击力+15%，HP-10%（速战速决环境）
  - **时空裂缝**（TIME_RIFT）：战斗时间流速 ×1.3（倒计时更快），所有冷却时间缩短 20%
  - 星图上异象星球显示对应图标（小浮动符号），PlanetPanel 详情中展示异象效果说明
  - 玩家可利用异象制定策略（如在 DARK_MATTER 星域派高 HP 舰队，在 NEBULA 星域用近战舰型）
- **文件**：`GalaxyScene.lua`（异象生成+星图图标渲染）、`BattleScene.lua`（异象效果逻辑）、`PlanetPanel.lua`（异象说明展示）、`Shared.lua`（异象定义表）
- **规模**：中（约 160 行）

#### P2-2：黑市走私网络

- **问题**：资源获取路径单一（建筑产出+战斗掉落+市场交易），中后期资源充裕后经济系统失去张力
- **方案**：星图解锁第 5 颗星球后开启"**黑市**"系统：
  - **黑市入口**：星图边缘出现"黑市空间站"标记（灰色骷髅图标），点击进入黑市面板
  - **走私货物**（每局随机刷新 3 种，60s 刷新一次）：
    - 违禁科技碎片：消耗 500 金属，获得当前未研究的随机 Tier2 科技（风险：15% 概率被截获，损失货物+扣除外交好感 10）
    - 军火箱：消耗 300 能源，获得一艘随机舰型（已满编时转为残骸零件×10）
    - 情报文件：消耗 200 晶体，揭示下 3 波海盗具体编成（完全预知）
    - 走私燃料：消耗 400 金属，所有远征时间减半，持续 120s
    - 赃物拍卖：消耗 100 各资源，随机获得 1 个模块（可能重复）
  - **截获风险**：每次交易有独立概率被"星际巡逻队"截获，截获后：
    - 该货物交易取消（资源不退）
    - 所有中立势力好感度-5
    - 连续被截获 3 次 → 黑市关闭 120s（"热度冷却"）
  - **走私达人成就**："黑市常客"（累计交易 20 次）、"完美走私"（连续 10 次未被截获）
- **文件**：`Client.lua`（黑市刷新+交易+截获判定）、`GalaxyScene.lua`（黑市空间站渲染）、`GameUI.lua`（黑市面板 UI）、`AchievementSystem.lua`（新成就）
- **规模**：中（约 170 行）

#### P2-3：阵型战术系统

- **问题**：舰队进入战斗后自动排列，玩家无法通过站位选择影响战斗结果；缺乏"排兵布阵"的策略感
- **方案**：战斗开始前（备战期内）可选择**舰队阵型**（默认"标准"，无加成）：
  - **锋矢阵**（WEDGE）：前排舰攻击力+20%，后排舰受击概率-30%；适合精锐少量舰队
  - **圆环阵**（CIRCLE）：所有舰船受击均匀分摊（等效 HP 池），ENGINEER 修复效率+50%；适合消耗战
  - **散兵阵**（SCATTER）：AOE 伤害（连锁/爆炸）对己方减半，但无法触发"协同护卫"被动；适合对抗 Boss
  - **冲锋阵**（CHARGE）：全舰攻速+30%，但 HP-15%；适合速战速决（配合 DARK_MATTER 异象更强）
  - 阵型选择 UI：备战期屏幕底部出现 4 个阵型图标按钮（带 tooltip 说明），选中后舰船位置动画重排
  - 切换阵型只能在备战期，战斗中锁定
  - 敌方也有阵型（海盗默认标准阵，Boss 波使用特殊阵型），宿敌会根据玩家历史阵型选择克制阵型
- **文件**：`BattleScene.lua`（阵型选择 UI + 位置计算 + 属性修正）、`Shared.lua`（阵型定义表）
- **规模**：中（约 140 行）

### P3 系列：体验打磨（低优先级）

#### P3-1：战斗回放与精彩时刻

- **问题**：精彩的连锁爆炸、极限翻盘瞬间转瞬即逝，玩家无法回顾；缺乏"炫耀时刻"的社交传播价值
- **方案**：战斗结算后新增"**回放精彩时刻**"按钮：
  - 系统自动标记 3 个"精彩瞬间"（连锁×3+、Boss 击杀、极限存活<10HP 翻盘）
  - 回放以 0.5× 慢速播放标记片段（每段 3s），镜头自动聚焦关键舰船
  - 回放时隐藏 HUD，仅显示伤害数字和特效
  - 回放结束后显示本战"MVP 舰船"（伤害最高/存活最久/治疗最多）
  - 技术实现：战斗过程中每帧记录关键状态（舰船位置/HP/事件），回放时重现
  - 存储：仅保留最近 1 场回放数据（内存中，不写入存档，避免数据膨胀）
- **文件**：`BattleScene.lua`（战斗状态录制+回放渲染，约 120 行）、`EndGamePanel.lua`（回放按钮+MVP 展示）
- **规模**：中（约 140 行）

#### P3-2：动态星图天气系统

- **问题**：星图始终是静态黑色背景+星球图标，视觉上缺乏"活着的宇宙"感受；长时间注视容易审美疲劳
- **方案**：星图新增**动态天气层**（纯视觉 + 轻度玩法影响）：
  - **星云漂移**：背景半透明彩色星云缓慢飘动（正弦运动），每 120s 换色
  - **流星雨**：随机方向的光点轨迹（每 30s 触发一次，持续 5s），纯装饰
  - **太阳风暴警告**：每 180s 随机选择 1 颗星球标记"太阳风暴来袭"（橙色脉冲圈），该星球下一波海盗进攻提前 15s
  - **空间裂隙闪烁**：宿敌即将出现前 30s，其目标星球附近出现紫色裂隙闪烁（预警暗示）
  - 天气效果用 NanoVG 粒子绘制（轻量级，不影响性能），所有效果可在设置中关闭
- **文件**：`GalaxyScene.lua`（天气粒子系统+预警渲染，约 100 行）、`SettingsPanel.lua`（天气开关选项）
- **规模**：小-中（约 110 行）

#### P3-3：快捷操作与生活质量包

- **问题**：游戏进入中后期（5+ 星球、多编队远征）后，重复性操作增多（逐个升级建筑、手动重复远征），操作效率成为体验瓶颈
- **方案**：一组 QoL 改进打包：
  - **远征重复**：远征结算面板新增"再次远征"按钮（相同目标、相同编队，一键出发）
  - **资源阈值警报**：设置面板可设定资源下限阈值（如金属<500 时提醒），低于阈值时顶栏对应资源图标闪烁红色
  - **一键全部升级**：帝国总览面板新增"批量升级"——对所有满足资源条件的建筑执行升级（带确认弹窗显示总消耗）
  - **快速造舰**：FleetPanel 长按造舰按钮 1s → 进入连续建造模式（每 0.5s 自动建造 1 艘，松手停止）
  - **战况速览**：战斗中按 TAB 显示简易统计面板（当前 DPS/总伤害/存活数/波次进度），松手消失
- **文件**：`FleetPanel.lua`（远征重复+快速造舰）、`SettingsPanel.lua`（阈值设定）、`GameUI.lua`（资源警报闪烁+帝国批量升级）、`BattleScene.lua`（TAB 速览面板）
- **规模**：中（约 130 行）

### P1 系列：核心玩法拓展（高优先级）

#### P1-1：派系外交深化（Faction Diplomacy）

- **问题**：现有 3 个中立势力仅提供好感度+协议的单维互动，缺乏阵营间的博弈张力——与 A 势力交好对 B 无影响，外交决策无代价
- **方案**：外交系统升级为**三角博弈**，势力间存在利益冲突：
  - **派系对立关系**：3 势力两两存在"竞争/中立/合作"关系（每局随机生成），与一方好感度过高（>70）会自动降低其竞争对手好感度-5/次
  - **外交手段扩展**（4 种新协议）：
    - 情报共享协议：好感≥40 解锁，揭示目标势力"讨厌"的对手下一步行动（提前知道海盗波次增援来自哪个方向）
    - 军事同盟协议：好感≥75 解锁，该势力在 Boss 波派出 2 艘援军参战（持续到 Boss 击败），但另外两势力好感各-10
    - 封锁禁令提案：好感≥50，花费 300 晶体发起对目标势力的贸易封锁，封锁期间黑市该势力货物消失（持续 180s）
    - 调停斡旋：好感≥60，花费 500 金属修复两个势力间的关系（竞争→中立），获得双方各+5 好感
  - **外交事件**（每 180s 随机触发一次）：
    - 势力请求：某势力要求你选边站（二选一：满足则+15/拒绝则-10）
    - 叛变警告：与你好感<20 的势力联合海盗发动突袭（下一波海盗规模+30%）
    - 贸易争端：两势力要求你仲裁，选择偏袒方+10/-5，中立裁决双方各+3
  - **UI**：星图中势力图标间新增关系连线（绿色=合作，红色=竞争，灰色=中立），外交面板新增"关系网"标签页
- **文件**：`Client.lua`（派系关系生成+外交事件逻辑+协议执行）、`GalaxyScene.lua`（关系连线渲染）、`GameUI.lua`（外交面板扩展）、`Systems.lua`（FactionRelation 数据结构）
- **规模**：大（约 250 行）

#### P1-2：终局危机事件链（Endgame Crisis）

- **问题**：游戏中后期（15+波次或无尽第 20 层之后）节奏趋于平稳，缺乏一个"改变一切"的史诗时刻；所有威胁来源只有海盗，玩法天花板到达后缺乏惊喜
- **方案**：新增**终局危机**——一个跨波次、全局性的末日事件链：
  - **触发条件**：困难模式第 12 波后 / 无尽模式第 25 层后，50% 概率触发（仅触发一次/局）
  - **3 种危机（随机 1 种）**：
    - **虚空蜂群**（VOID_SWARM）：外星入侵，额外生成"虚空"编队从星图边缘进攻所有星球；每 3 波出现一次虚空小队（4 艘高攻低HP），持续 5 波后撤退
    - **叛逆 AI**（ROGUE_AI）：你的 1 颗随机殖民星球"叛变"——该星球建筑全部停产，变为敌方据点，生成 AI 守军；需派远征军夺回（远征战力要求 ×1.5）
    - **暗能风暴**（DARK_STORM）：全星图每 15s 对随机 1 颗星球施加"暗能腐蚀"（建筑产量-50%持续 30s），可通过消耗 200 晶体"净化"解除；持续 4 分钟后自然消散
  - **危机通知**：触发时全屏橙色警告横幅"⚠ 终局危机：[名称]"持续 3s，星图对应位置出现脉冲警告图标
  - **危机奖励**：成功存活/解决危机后获得：
    - 文明积分 ×5（大幅推进传承进化）
    - 专属"危机幸存者"称号（排行榜显示）
    - 残骸零件 ×15
  - **新成就**："危机管理者"（首次存活过终局危机）、"无畏指挥官"（在终局危机期间不损失任何星球）
- **文件**：`Client.lua`（危机触发判定+状态管理+奖励结算）、`GalaxyScene.lua`（危机视觉警告+虚空编队渲染）、`BattleScene.lua`（虚空蜂群战斗逻辑）、`AchievementSystem.lua`（新成就）
- **规模**：大（约 280 行）

#### P1-3：指挥官系统（Commander Officers）

- **问题**：舰队管理维度仅有"舰船数量+模块选择"，缺乏RPG式的"人才培养"体验；编队间除数量外没有个性差异
- **方案**：新增**指挥官**角色，每个编队可分配 1 名指挥官，提供独特加成：
  - **获取方式**：
    - 战役序章完成奖励 1 名初始指挥官（"新星"级）
    - 击败宿敌最终决战获得宿敌转化指挥官（继承宿敌特殊能力）
    - 每日挑战连续完成 7 天奖励随机指挥官
    - 黑市偶尔出售"雇佣兵指挥官"（一次性，本局有效）
  - **指挥官属性**：
    - 等级 1-10（每场战斗胜利获得经验，经验=击杀数×2+存活波次×5）
    - 专精类型（3 选 1，Lv3 时锁定）：
      - 战术专精：编队攻击力+2%/级，Lv5 解锁主动技能"精确轰击"（瞬间对敌方血量最高舰船造成 200 伤害，90s CD）
      - 防御专精：编队 HP+3%/级，Lv5 解锁"紧急护盾"（全编队获得 15% HP 等值护盾，120s CD）
      - 后勤专精：资源获取+1.5%/级，Lv5 解锁"战场回收"（击败敌舰时 25% 概率获得残骸零件，被动）
  - **指挥官面板**：FleetPanel 编队卡片顶部显示指挥官头像+等级徽章；点击头像打开详情面板（经验条、专精选择、技能说明）
  - **跨局保留**：指挥官写入 `galaxy_career.json`（与传承树同级），永久保留等级和专精
  - **上限**：最多同时拥有 4 名指挥官（多余的可"退役"换取文明积分×3）
- **文件**：`Client.lua`（指挥官数据管理+经验计算+存档）、`FleetPanel.lua`（指挥官面板UI）、`BattleScene.lua`（指挥官技能触发+加成应用）、`Systems.lua`（Commander 数据结构定义）
- **规模**：大（约 260 行）

### P2 系列：玩法深度扩展（中优先级）

#### P2-1：程序化任务板（Procedural Quest Board）

- **问题**：游戏目标来源仅有"守住海盗进攻"和"阶段目标15条"，玩家在中期完成大部分目标后缺乏短期引导；探索任务结束后无后续
- **方案**：银河界面新增"**任务板**"系统（每 120s 刷新 1 个新任务，最多同时 3 个活跃任务）：
  - **任务类型**（按权重随机生成）：
    - 歼灭任务（40%）："在 N 秒内击败下一波海盗" / "本波战斗中不损失舰船" / "用 [阵型] 赢得一场战斗"
    - 经济任务（30%）："积累 X 金属" / "在黑市完成一次交易" / "远征 [目标] 并获胜"
    - 外交任务（20%）："与 [势力] 好感度达到 X" / "完成一次调停斡旋" / "签订军事同盟"
    - 探索任务（10%）："殖民一颗带 [异象] 的星球" / "在 [天气] 条件下作战"
  - **奖励分级**：
    - 普通任务：金属/能源 100-300 + 残骸零件 1-2
    - 精英任务（20% 概率生成，带金色边框）：资源 ×3 + 指挥官经验 ×2 + 随机模块
  - **任务失败**：超时（任务有 300s 限时）或条件违反则自动失败并移除，无惩罚
  - **UI**：银河界面左下角"📋"图标，点开显示任务列表（最多3条），完成时顶部弹出奖励横幅
- **文件**：`Client.lua`（任务生成+条件检测+奖励发放）、`GameUI.lua`（任务板面板+完成通知）
- **规模**：中（约 180 行）

#### P2-2：巨构工程（Megastructure Projects）

- **问题**：星球建筑体系在中后期完全建满后失去成长感，缺乏"投入大量资源换取巨大回报"的长线目标
- **方案**：殖民 4+ 颗星球后解锁"**巨构工程**"系统——跨星球的超级建筑项目：
  - **3 种巨构**（每局仅可建造 1 个，需先选择再投入）：
    - **戴森球**（Dyson Sphere）：
      - 消耗：金属 3000 + 能源 2000 + 晶体 1000，分 5 阶段建造（每阶段 120s）
      - 效果：建成后所有星球能源产量 +100%，战斗中ENGINEER修复频率翻倍
    - **星门网络**（Stargate Network）：
      - 消耗：金属 2000 + 晶体 2500 + 核能 1500，分 4 阶段
      - 效果：远征时间归零（瞬间到达）、编队可在任意星球间瞬移、海盗进攻预警时间 +30s
    - **量子堡垒**（Quantum Fortress）：
      - 消耗：金属 4000 + 核能 2000，分 6 阶段
      - 效果：所有战斗中初始获得全编队 20% HP 等值护盾、终局危机伤害减半、Boss 波额外获得 2 颗轨道炮（自动攻击最近敌舰）
  - **建造进度**：帝国总览面板顶部显示巨构进度条（分阶段节点），每阶段完成有里程碑动画
  - **战略意义**：巨构选择是不可逆的全局决策——军事型/经济型/后勤型对应不同的游戏策略
  - **新成就**："宇宙建筑师"（建成任意巨构）
- **文件**：`Client.lua`（巨构状态管理+阶段推进+效果应用）、`GameUI.lua`（巨构进度UI+选择面板）、`BattleScene.lua`（量子堡垒战斗效果）、`AchievementSystem.lua`（新成就）
- **规模**：中-大（约 200 行）

#### P2-3：舰队涂装与徽章（Fleet Cosmetics）

- **问题**：所有玩家的舰队在视觉上完全相同（颜色由舰型决定），缺乏个性化表达；排行榜上无法一眼辨识个人风格
- **方案**：新增**舰队涂装**系统——纯视觉定制，不影响数值：
  - **涂装色板**（12 种主色 + 12 种辅色组合）：
    - 解锁条件：初始 3 套免费，其余通过成就/联赛段位/危机存活/宿敌击杀解锁
    - 应用范围：影响己方所有舰船在战斗中的色调（主色=舰身，辅色=引擎尾焰）
  - **军旗徽章**（20 种图案）：
    - 类型：几何图形（星/盾/剑/翼/环等）、成就专属（危机幸存者骷髅/联赛冠军王冠）、宿敌战利品（击败宿敌后获得其徽章）
    - 展示位置：排行榜名称前图标、战斗中旗舰上方小标志、结算页个人信息栏
  - **涂装面板**：设置面板新增"涂装"标签页，预览效果 + 一键应用
  - **存储**：涂装选择写入 `galaxy_career.json`，跨局保持
- **文件**：`Client.lua`（涂装存档+加载）、`SettingsPanel.lua`（涂装选择面板）、`BattleScene.lua`（舰船颜色应用）、`GameUI.lua`（排行榜徽章渲染）
- **规模**：中（约 150 行）

### P3 系列：体验打磨（低优先级）

#### P3-1：银河百科（Galactopedia）

- **问题**：游戏系统已非常丰富（改装/异象/阵型/外交/黑市/宿敌/危机/巨构/指挥官），新玩家和回归玩家面对大量机制缺乏查阅途径，只能靠记忆
- **方案**：主菜单新增"**百科**"入口（📖图标），内含全部游戏机制的结构化百科：
  - **分类标签**（6 大类）：
    - 舰种图鉴：8 种舰型属性+被动+推荐模块+推荐阵型
    - 模块大全：15 种改装模块效果+获取方式+推荐搭配
    - 星域手册：6 种异象效果+对策建议
    - 外交指南：派系关系机制+协议类型+外交事件应对
    - 危机档案：3 种终局危机详情+存活策略
    - 指挥官名录：所有已获得指挥官详情+专精对比
  - **解锁机制**：百科条目在玩家首次遇到对应内容时自动解锁（灰色→彩色），提供"全解锁"的收集驱动
  - **UI**：全屏面板，左侧分类导航，右侧卡片式条目详情；已解锁进度条显示在顶部（如 "45/68 条已解锁"）
- **文件**：`GameUI.lua`（百科面板渲染+分类导航+条目详情）、`Client.lua`（解锁状态追踪+存档）
- **规模**：中（约 160 行）

#### P3-2：战报导出与社交分享

- **问题**：V2.3 加入了战斗回放精彩时刻，但仅限游戏内观看，无法导出或分享；排行榜战绩缺乏对外展示途径
- **方案**：结算页面新增"**生成战报**"按钮，生成结构化文本战报：
  - **战报内容**：
    - 标题行："🏆 银河征服 - [难度] 第 [N] 波 存活！" / "💀 第 [N] 波 惜败"
    - 核心数据：存活波次、总击杀、DPS峰值、MVP舰船、使用阵型
    - 精彩时刻摘要（最多3个）："第7波 连锁爆炸×4！" / "Boss 极限翻盘（旗舰仅剩8HP）"
    - 舰队配置快照：舰型×数量 + 指挥官名称 + 巨构进度
    - 联赛/挑战标签（如适用）
  - **生成方式**：点击按钮 → 战报文本自动复制到系统剪贴板 + 屏幕短暂显示"已复制到剪贴板 ✓"
  - **排行榜集成**：排行榜页面长按他人条目 → 显示该玩家最佳战报摘要（来自 cloudScore extra 字段，限 200B）
- **文件**：`EndGamePanel.lua`（战报生成+复制逻辑）、`GameUI.lua`（排行榜战报预览）
- **规模**：小-中（约 100 行）

#### P3-3：性能优化与辅助功能

- **问题**：随着系统复杂度增加（天气粒子+异象效果+宿敌动画+巨构进度），低端设备帧率可能出现瓶颈；同时缺乏色觉障碍支持
- **方案**：一组性能和辅助功能打包：
  - **性能档位**：设置面板新增"画质"选项（高/中/低）：
    - 低：关闭天气粒子、卫星公转、流星雨、战斗尾焰特效
    - 中：保留天气但粒子数量减半、尾焰简化为单色
    - 高：全特效（默认）
  - **色觉辅助模式**：
    - 设置新增"色觉辅助"开关，开启后：
    - 所有红/绿区分改为红/蓝区分（敌方=红→橙，己方=绿→蓝）
    - 资源颜色增加图案纹理区分（不仅靠颜色）
    - 危机/威胁警告增加闪烁频率标识
  - **帧率显示**：设置新增"显示FPS"开关，左上角显示实时帧率
  - **自动暂停**：切到后台时自动暂停游戏（银河界面），返回后显示"已暂停"提示
- **文件**：`SettingsPanel.lua`（画质选项+色觉开关+FPS开关）、`Client.lua`（画质配置应用+自动暂停）、`GalaxyScene.lua`（粒子裁剪按画质档位）、`BattleScene.lua`（特效简化逻辑）
- **规模**：中（约 130 行）

### 设计方向

V2.4 完成后，游戏已具备完整的**战略-战术-养成-社交**闭环。V2.5 聚焦三个主题：

1. **深度重玩性** — 让每局游戏体验更独特（星系生成器、变异舰船、文明遗产）
2. **玩家表达** — 给予更多自定义/创造空间（自定义阵型、战术蓝图、舰队命名）
3. **代码健康** — 对超 5000 行文件执行结构化拆分，解决 local 变量溢出根因

### P1 系列：核心深度（高优先级）

#### P1-1：星系生成器（Galaxy Procedural Seeds）

- **问题**：每局银河地图结构相似（固定星球数+随机位置），重玩 50+ 局后缺乏新鲜感；当前联赛已有种子系统但仅影响难度修正，未触及地图结构
- **方案**：引入完整的**程序化星系生成器**，每局根据 seed 生成结构迥异的银河：
  - **星系形态**（5 种，seed 决定）：
    - 螺旋星系（默认）：6-8 颗行星，均匀分布
    - 星团：10 颗行星密集于中心区域，距离近但资源争夺激烈
    - 虚空走廊：4 颗行星呈线性分布，远征路径唯一但漫长
    - 双子星系：两组 4 颗行星，中间空旷区有唯一跳跃点
    - 环形星系：8 颗行星环绕中心虚空，中心有 1 颗高价值"核心星"
  - **特殊星球类型**（seed 概率触发）：
    - 死星（Dead World）：无法殖民，但每 3 波产出稀有残骸可回收
    - 风暴星（Storm World）：产出高但每 60s 有 10% 概率触发风暴事件（-30% 产出 20s）
    - 双子星（Binary World）：2 颗行星共享轨道，殖民一颗送另一颗低产基地
  - **Seed 输入**：联赛使用固定周 seed；普通模式每局随机 seed 可手动输入/分享
  - **UI**：开始新局时显示星系预览缩略图 + seed 码（6 位字母数字）
- **文件**：新增 `game/GalaxyGenerator.lua`（生成逻辑）；修改 `GalaxyScene.lua`（接入生成器替代硬编码布局）、`Client.lua`（seed 存档/联赛 seed 传递）
- **规模**：中（约 200 行新增 + 80 行修改）

#### P1-2：变异舰船（Mutant Ships）

- **问题**：8 种舰船基础属性固定，改装模块主要是数值提升，缺乏"独一无二的舰船"体验；Boss 波次后奖励也已模式化
- **方案**：引入**变异舰船**机制——战斗中低概率获得带有永久变异词缀的特殊舰船：
  - **获取方式**：
    - Boss 击杀后 15% 概率掉落变异舰船（替代普通舰船奖励）
    - 深空异象"太古残骸"事件：50% 概率获得 1 艘变异舰船
    - 程序化任务"基因收集"完成奖励
  - **变异词缀**（共 12 种，每艘附带 1-2 个）：
    - 正面（8 种）：吸血（击杀回 5%HP）/ 分裂（死亡时生成 2 艘微型副本）/ 狂暴（HP<30% 攻击×2）/ 隐形（每 20s 隐身 3s）/ 再生（每秒回 1%HP）/ 过载（攻速+50% 但持续扣 HP）/ 感电（攻击附带范围溅射）/ 守护（相邻友军减伤 15%）
    - 负面（4 种，组合降低获取难度）：脆弱（-20%HP）/ 迟缓（移速-30%）/ 耗能（维护费×2）/ 不稳定（每 30s 随机失控 2s）
  - **编队规则**：每编队最多装备 2 艘变异舰船；变异舰船不可改装（词缀替代模块）
  - **视觉**：变异舰船名称前缀带 ⚡ 标记，战斗中船体有微弱电弧特效
  - **新成就**："基因收藏家"（拥有 5 艘不同词缀的变异舰船）
- **文件**：新增 `game/MutantShipSystem.lua`（变异生成/词缀效果/存储）；修改 `BattleScene.lua`（词缀战斗效果触发）、`FleetPanel.lua`（变异舰船展示）、`Client.lua`（存档）、`AchievementSystem.lua`（新成就）
- **规模**：大（约 280 行新增 + 120 行修改）

#### P1-3：文明遗产（Civilization Legacy）

- **问题**：跨局只有联赛积分、指挥官、涂装解锁等"账户级"持久要素，缺乏"文明发展"的宏大感——每次开新局像从零开始的陌生人
- **方案**：新增**文明遗产**系统——跨局积累的永久微增益，体现"银河文明的崛起"：
  - **遗产点**（Legacy Points, LP）获取：
    - 每局结束时根据表现奖励 LP：存活 10 波 = 5LP，击杀 100 = 3LP，建成巨构 = 10LP，危机存活 = 15LP，联赛前 3 = 8LP
    - LP 上限 999（防止无限叠加变质）
  - **遗产树**（3 条分支，每条 5 级）：

### P1 系列：社交深度（高优先级）

#### P1-1：好友系统 2.0

- **问题**：现有 FriendSystem (460行) 仅支持好友列表，缺乏互动和激励机制
- **方案**：
  - **好友助战**：可邀请好友的"传奇舰船"（好友历史最强舰船）参战 1 次/天
  - **友谊赛**：好友间可发起挑战赛（限定舰船数量，胜负不计入排名）
  - **礼物系统**：每日可向好友赠送资源礼物（无消耗，仅增友谊值），友谊值达 100 解锁称号
  - **好友战报**：可查看好友最近战绩（简化版战报）
- **文件**：`systems/FriendSystem.lua`（扩展）+ `ui/FriendPanel.lua`（新增礼物/友谊赛 UI）
- **规模**：中（约 180 行）

#### P1-2：公会系统扩展

- **问题**：GuildSystem (661行) 仅支持基础成员管理，缺乏公会活动和目标感
- **方案**：
  - **公会战**：每周 1 次公会战（3v3 自动对抗，3 场战斗总分定胜负），参与奖励公会币
  - **公会任务**：每日 3 个公会任务（击败 N 艘敌舰/殖民 N 颗星球等），全员共享进度
  - **公会商店**：使用公会币兑换专属奖励（传说模块碎片/指挥官经验药水/涂装）
  - **公会日志**：记录成员活动、贡献度、战果
  - **会长权限**：踢人、设置公告、任命官员
- **文件**：`systems/GuildSystem.lua`（扩展）+ `ui/GuildPanel.lua`（扩展）+ 新增公会战逻辑
- **规模**：中-大（约 250 行）

#### P1-3：赛季系统深化

- **问题**：SeasonSystem (522行) 和 SeasonExpansionSystem 已有赛季概念，缺乏跨赛季叙事和赛季专属奖励
- **方案**：
  - **赛季主题**：每个赛季赋予独特主题（"星际争霸"/"银河风暴"/"虚空入侵"），影响 BGM、星球装饰、特殊事件
  - **赛季通行证**：免费档 + 付费档（赛季积分兑换），付费档含专属皮肤/指挥官/涂装
  - **赛季任务**：10 个赛季专属任务（如"使用变异舰船赢得 5 场战斗"），完成后获得赛季币
  - **赛季排行榜**：独立赛季积分排行榜，前 100 名获得专属头像框
  - **赛季商店**：使用赛季币兑换往期奖励（补齐错过内容）
- **文件**：`systems/SeasonSystem.lua`（扩展）+ `ui/SeasonShopPanel.lua`（扩展）
- **规模**：中-大（约 220 行）
