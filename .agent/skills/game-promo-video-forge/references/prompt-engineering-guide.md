## 1. Prompt 结构黄金公式

### 基本公式

```
[主体描述] + [动作/运动] + [运镜方式] + [光影/氛围] + [风格修饰]
```

### 各要素说明

| 要素 | 作用 | 示例 |
|------|------|------|
| 主体描述 | 画面中最重要的对象 | "一位穿着银色铠甲的骑士" |
| 动作/运动 | 画面中发生的运动 | "挥舞巨剑斩向飞龙" |
| 运镜方式 | 镜头如何移动 | "镜头从低角度仰拍缓缓上移" |
| 光影/氛围 | 环境光线和情绪 | "夕阳逆光，金色光辉笼罩" |
| 风格修饰 | 画面整体风格 | "电影级画质，虚幻引擎风格渲染" |

### 完整示例

```
一位穿着银色铠甲的骑士在悬崖边挥舞巨剑，
镜头从低角度仰拍缓缓环绕上升，
夕阳逆光照射下铠甲反射金色光辉，
斗篷在风中飘动，电影级画质，史诗奇幻氛围
```

### 长度建议

| 场景类型 | 建议 prompt 长度 | 说明 |
|---------|-----------------|------|
| 简单场景 | 30-50 字 | Logo 展示、静态场景 |
| 标准场景 | 50-100 字 | Gameplay 展示、角色动作 |
| 复杂场景 | 100-200 字 | 多元素交互、电影级镜头 |

## 2. 游戏宣传视频 Prompt 模板

### 模板 A：世界观揭示

```
[描述游戏世界的宏大场景],
镜头从高空鸟瞰缓缓下降推进,
[光影氛围],
电影级画质, [风格关键词]
```

**填充示例**：
```
一片广袤的奇幻大陆从云层中浮现，古老的城堡矗立在悬崖之上，
魔法能量的光柱连接天地，镜头从高空鸟瞰缓缓下降推进到城堡大门前，
晨曦金色光芒穿透云层照亮大地，
电影级画质，虚幻引擎风格渲染，史诗奇幻氛围
```

### 模板 B：角色登场

```
[角色外观描述]站在[场景中],
[角色动作/姿态],
镜头[运镜方式],
[光影氛围], 角色设计精美, 高品质3D渲染
```

**填充示例**：
```
一位身穿暗红色法袍的女法师站在燃烧的废墟中，
双手举起凝聚一团紫色闪电魔法球，法袍和长发被魔力气场吹起飘动，
镜头从正面中景缓缓环绕到侧面，
暗红色火焰光映照面部，角色设计精美，高品质3D渲染
```

### 模板 C：战斗场面

```
[战斗场景描述],
[战斗动作和特效],
镜头[运镜], [速度描述],
[氛围], 动作流畅, 粒子特效绚丽
```

**填充示例**：
```
两个阵营的军队在平原上展开激烈对战，
剑与盾碰撞火花四溅，魔法箭雨从天空落下爆炸，
镜头快速跟拍一位冲锋的骑士，然后急速拉升到鸟瞰全景，
硝烟弥漫的战场氛围，动作流畅，粒子特效绚丽
```

### 模板 D：功能展示

```
游戏界面展示[功能名称],
[具体操作演示],
[UI/系统的视觉特征],
清晰的游戏画面, 流畅的交互动画
```

### 模板 E：Logo 结尾

```
[背景描述]中，游戏Logo[出现方式],
Logo文字为"[游戏名]",
[光效/粒子装饰],
简洁大气, 游戏品牌风格
```

**填充示例**：
```
深邃星空背景中，游戏Logo从光点汇聚凝结而成，
Logo下方文字"星际征途"闪烁发光，
周围星尘粒子环绕飘散，
简洁大气，科幻游戏品牌风格
```

## 3. 运镜语言参考

### 基础运镜

| 运镜 | Prompt 描述 | 适用场景 |
|------|------------|---------|
| 推进 (Dolly In) | "镜头缓缓推进靠近主体" | 聚焦角色/物品 |
| 拉远 (Dolly Out) | "镜头缓缓拉远展示全貌" | 揭示世界全景 |
| 平移 (Pan) | "镜头从左向右水平平移" | 展示宽阔场景 |
| 仰拍 (Low Angle) | "低角度仰拍，主体显得高大" | 表现角色威严 |
| 俯拍 (High Angle) | "高角度俯瞰，鸟瞰全景" | 展示地图/战场 |
| 跟拍 (Tracking) | "镜头跟随角色移动" | 动作/跑酷场景 |

### 进阶运镜

| 运镜 | Prompt 描述 | 适用场景 |
|------|------------|---------|
| 环绕 (Orbit) | "镜头360度环绕主体旋转" | 角色/产品展示 |
| 升降 (Crane) | "镜头从地面升起到高空" | 开场世界揭示 |
| 甩镜 (Whip Pan) | "镜头快速甩向一侧" | 场景快速切换 |
| 穿越 (Fly-through) | "镜头穿过门/窗飞入室内" | 探索室内场景 |
| 微距 (Macro) | "极近距离特写细节" | 道具/装备展示 |
| 手持 (Handheld) | "轻微晃动的手持风格" | 增加临场感 |

### 速度修饰

| 速度 | Prompt 关键词 | 适用场景 |
|------|-------------|---------|
| 极慢 | "超慢动作，时间仿佛静止" | 关键一击/爆炸 |
| 缓慢 | "缓缓移动" | 世界展示/氛围 |
| 正常 | "平稳移动" | 常规展示 |
| 快速 | "快速移动""急速" | 动作/追逐 |
| 变速 | "从慢逐渐加速" | 节奏渐强 |

## 4. 转场 Prompt 技巧

### 使用 first_last_frame 模式

当需要两个截图之间平滑过渡时，使用 `create_video_task` 的 `first_last_frame` 模式：

```
mode: "first_last_frame"
images: [
  { url: "截图A.png", role: "first_frame" },
  { url: "截图B.png", role: "last_frame" }
]
prompt: "[转场描述]"
```

### 转场 Prompt 示例

| 转场类型 | Prompt 描述 |
|---------|------------|
| 溶解 | "画面柔和渐变，从前一场景自然过渡到下一场景" |
| 推移 | "画面从左向右推移切换到新场景" |
| 缩放 | "镜头快速推近一个物体然后拉远展示新场景" |
| 闪白 | "画面突然闪亮白光然后展示新的场景" |
| 粒子 | "画面碎裂成无数粒子然后重新组合成新场景" |
| 旋转 | "画面以中心为轴旋转切换到新场景" |
| 黑幕 | "画面渐暗至全黑然后渐亮展示新场景" |

## 6. 常见问题与优化

### 问题 1：生成的视频动作太少/太静

**原因**：prompt 中缺少动作描述

**修复**：
```
❌ "一个骑士站在城堡前"
✅ "一个骑士在城堡前缓缓拔出长剑，斗篷被风吹起飘动，
    镜头缓缓环绕上升"
```

### 问题 2：画面与游戏风格不匹配

**原因**：缺少风格限定词

**修复**：
```
❌ "一个角色在森林中行走"
✅ "卡通风格的角色在彩色低多边形森林中行走，
    明亮的色彩，Q版造型，休闲游戏风格"
```

### 问题 3：Logo/文字不清晰

**原因**：AI 不擅长精确文字渲染

**修复方案**：
1. 使用 `generate_image` 先生成带 Logo 的静态图
2. 将图片作为 `first_frame` 传入 `create_video_task`
3. prompt 描述 Logo 周围的动态效果而非文字本身

### 问题 4：场景太短看不清内容

**原因**：duration 设置过短

**修复**：
- 展示型场景：≥5s
- 动作型场景：≥4s
- Logo/CTA：≥3s
- 转场衔接：≥2s

### 问题 5：多段视频风格不统一

**原因**：每段 prompt 风格描述不一致

**修复**：
- 在每段 prompt 末尾追加相同的风格锚定词
- 例如统一追加 "卡通渲染风格，明亮色彩，游戏宣传片品质"
- 使用 `seed` 参数让随机性可控

### 问题 6：视频有水印或模糊

**修复**：
- 检查 `resolution` 参数（建议 720p）
- 检查输入图片分辨率（建议 ≥1024px）
- 查询视频任务状态确认生成成功

## 8. Prompt 检查清单

在将 prompt 提交给 `create_video_task` 前，逐项检查：

- [ ] **主体明确**：描述了画面中最重要的对象
- [ ] **动作存在**：至少有一个动态元素（角色动作/镜头运动/环境变化）
- [ ] **运镜描述**：指定了镜头移动方式
- [ ] **风格一致**：追加了与游戏风格匹配的修饰词
- [ ] **长度合理**：30-200 字之间
- [ ] **mode 正确**：`text_to_video`（纯文字）/ `first_frame`（有图片）/ `first_last_frame`（有首尾帧）
- [ ] **duration 合理**：4-15s 之间
- [ ] **ratio 正确**：横屏 16:9 / 竖屏 9:16

### Prompt 质量评级

| 等级 | 特征 | 示例 |
|------|------|------|
| S 级 | 五要素齐全 + 品类风格精准 | "一位穿着银色铠甲的骑士在悬崖边挥舞巨剑，镜头从低角度仰拍缓缓环绕上升，夕阳逆光照射下铠甲反射金色光辉，斗篷在风中飘动，电影级画质，史诗奇幻氛围" |
| A 级 | 三要素以上 + 有风格修饰 | "骑士挥剑，镜头环绕，逆光，史诗风格" |
| B 级 | 有主体和动作，缺运镜 | "骑士在悬崖边战斗" |
| C 级 | 仅主体描述 | "一个骑士" |

**目标**：所有宣传视频 prompt 达到 A 级及以上。
