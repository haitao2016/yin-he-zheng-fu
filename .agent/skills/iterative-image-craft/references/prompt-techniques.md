# 高级 Prompt 技巧与常见问题

> 本文档是 `iterative-image-craft` Skill 的补充参考，提供图像生成/编辑的 Prompt 编写技巧。

---

## 六维度 Prompt 构建法

### 维度详解

#### 1. 主体（Subject）

描述画面的核心对象，要具体明确。

| 差 | 好 | 更好 |
|---|---|------|
| "一个人" | "一个战士" | "一个穿着银色铠甲的女战士，手持长剑" |
| "一只动物" | "一只猫" | "一只橘色虎斑猫，蜷缩着睡觉" |
| "一个东西" | "一把剑" | "一把发光的蓝色水晶长剑，剑身有符文" |

**技巧**：
- 先说"是什么"，再说"什么样的"
- 描述关键特征（颜色、材质、姿态、表情）
- 游戏角色注意描述装备和动作

#### 2. 风格（Style）

指定艺术风格，直接影响整体视觉感受。

| 风格关键词 | 效果 | 适用场景 |
|-----------|------|---------|
| 像素风格/pixel art | 复古像素画 | 复古游戏、独立游戏 |
| 卡通风格 | 圆润可爱 | 休闲游戏、儿童向 |
| 写实风格 | 接近照片 | 模拟经营、策略游戏 |
| 水彩风 | 柔和晕染 | 文艺游戏、视觉小说 |
| 赛博朋克 | 霓虹、科技感 | 科幻游戏 |
| 日式动漫 | 二次元风格 | RPG、视觉小说 |
| 扁平化 | 简洁矢量感 | UI 元素、图标 |
| 暗黑哥特 | 阴暗沉重 | 恐怖、暗黑类游戏 |

#### 3. 构图（Composition）

控制视角和画面布局。

| 关键词 | 效果 |
|-------|------|
| 正面居中 | 对称正面视角（适合图标、头像） |
| 正面特写 | 面部/上半身特写 |
| 全身站立 | 角色全身展示 |
| 俯视 / 鸟瞰 | 从上往下看（适合地图、场景） |
| 侧面 / 侧视 | 横版游戏角色 |
| 45度角 | 等距视角（适合策略游戏） |
| 广角全景 | 宽阔场景 |

#### 4. 光照（Lighting）

控制光线效果和氛围。

| 关键词 | 效果 | 氛围 |
|-------|------|------|
| 柔和自然光 | 均匀柔和 | 温馨、日常 |
| 强烈阳光 | 高对比 | 活力、冒险 |
| 霓虹灯光 | 彩色光晕 | 科幻、夜景 |
| 逆光剪影 | 暗主体亮背景 | 神秘、史诗 |
| 烛光 | 暖黄微弱 | 中世纪、温暖 |
| 月光 | 冷蓝柔和 | 夜晚、宁静 |
| 无特定光照 | AI 自动决定 | 通用 |

#### 5. 背景（Background）

描述主体后方的环境。

| 关键词 | 说明 |
|-------|------|
| 透明背景 | 适合图标、角色立绘（需设 `transparent: true`） |
| 纯色背景 | "白色背景"、"黑色背景" |
| 场景背景 | "森林"、"城堡"、"太空"、"海底" |
| 渐变背景 | "蓝紫渐变背景" |
| 模糊背景 | "背景虚化"（突出主体） |

#### 6. 色调（Color Tone）

控制整体色彩倾向。

| 关键词 | 效果 |
|-------|------|
| 暖色调 | 橙黄为主，温暖舒适 |
| 冷色调 | 蓝紫为主，冷静科技 |
| 高饱和 | 色彩鲜艳夺目 |
| 低饱和 / 淡雅 | 柔和不刺眼 |
| 单色 / 黑白 | 只有一种色系 |
| 复古色调 | 褪色偏黄 |
| 霓虹色 | 高对比荧光色 |

## 常见 Prompt 模板

### 游戏图标

```
"{物品名称}图标，{风格}，正面居中，{光照}，透明背景，{色调}"
```

示例：
- "金币图标，像素风格，正面居中，金属光泽，透明背景，高饱和金色"
- "红色药水瓶图标，卡通风格，正面居中，柔和光照，透明背景，暖色调"
- "闪电技能图标，扁平化风格，居中，发光效果，深色背景，蓝紫霓虹"

### 角色立绘

```
"{角色描述}，{风格}，{视角}，{光照}，{背景}，{色调}"
```

示例：
- "穿红色铠甲的女骑士，日式动漫风格，全身站立，柔和自然光，透明背景，暖色调"
- "戴兜帽的神秘法师，暗黑哥特风格，半身特写，烛光，黑色背景，冷蓝色调"

### 场景背景

```
"{场景描述}，{风格}，{视角}，{光照}，{色调}"
```

示例：
- "奇幻森林，水彩风，广角全景，阳光透过树叶，翠绿色调"
- "赛博朋克都市夜景，写实风格，鸟瞰俯视，霓虹灯光，蓝紫霓虹色"

### 纹理贴图

```
"{材质描述}，无缝贴图，平铺纹理，{色调}"
```

示例：
- "石砖墙面，无缝贴图，平铺纹理，灰褐色调"
- "草地纹理，无缝贴图，俯视平铺，翠绿色调"

## 迭代调整技巧

### 微调 Prompt 的方法

| 问题 | 调整方向 | 示例 |
|------|---------|------|
| 太暗 | 加亮度/光照关键词 | "明亮的"、"强烈阳光"、"高亮度" |
| 太亮 | 加暗调关键词 | "暗色调"、"低光照"、"阴影" |
| 色彩太淡 | 加饱和度 | "高饱和"、"鲜艳"、"浓郁色彩" |
| 色彩太浓 | 减饱和度 | "淡雅"、"柔和色调"、"低饱和" |
| 细节不够 | 加细节描述 | 具体描述想要的细节部分 |
| 太复杂 | 简化描述 | 去掉多余关键词，保留核心 |
| 风格不对 | 更换/强调风格词 | 加"纯粹的..."、"标准的..." |
| 构图不对 | 明确视角 | "正面"、"45度角"、"俯视" |

### seed 的使用

- **同一 prompt + 不同 seed**：生成同风格的不同变体，用于对比选择
- **同一 prompt + 相同 seed**：复现完全相同的结果（调试用）
- **不指定 seed**：每次随机，适合探索阶段

### reference_images 技巧

1. **风格锚定**：先确认一张满意的图，作为后续所有图的 reference
2. **多参考混合**：提供 2-3 张参考图，AI 会综合它们的特点
3. **角色一致性**：用正面图做 reference，生成同角色不同姿势
4. **注意**：reference_images 最多 14 张，edit_image 最多 13 张（原图占 1 张）

## 常见问题与解决

### Q: 透明背景没生效？

确保同时满足：
1. prompt 中包含"透明背景"
2. 设置 `transparent: true` 参数

### Q: 生成的图和描述差别很大？

- 简化 prompt，不要堆砌太多关键词
- 把最重要的描述放在 prompt 开头
- 尝试 `thinking_level: "high"` 让 AI 更深度理解

### Q: 编辑后角色/物体变形了？

- 编辑 prompt 要具体指明"保持角色/物体不变"
- 使用 `reference_images` 传入原图作为参考
- 编辑范围越小，保真度越高

### Q: 多张图风格不一致？

- 用第一张确认的图作为后续所有图的 `reference_images`
- 保持 prompt 中风格关键词完全一致
- 考虑使用 `seed` 固定随机因子

### Q: 纹理贴图有明显拼接痕迹？

- prompt 中加"无缝"、"可平铺"关键词
- 生成后需要在游戏引擎中实际测试平铺效果
- 必要时用 `edit_image` 修复边缘

### Q: 迭代超过 5 轮还不满意？

- 停下来重新审视需求：是否描述不够准确？
- 提供参考图（现有素材或网络图片）
- 考虑调整需求方向而非继续硬磨
- 换一种风格关键词，可能打开新思路
