# 歌词写作与 AI 音乐歌词工程参考

> AI 音乐制作 prompt 工程专用参考。涵盖歌词结构、韵律技巧、中文歌词写作与 AI 歌词标签。

---

## 1. 歌词段落功能

| 段落 | 英文名 | 功能 | 情感弧线 | 歌词特征 |
|------|--------|------|---------|---------|
| 前奏 | Intro | 建立氛围 | 引入 | 可无词或哼唱 |
| 主歌A | Verse 1 | 叙事铺垫 | 展开 | 细节描写，故事背景 |
| 主歌B | Verse 2 | 深化推进 | 递进 | 视角转换，深层情感 |
| 预副歌 | Pre-Chorus | 过渡积蓄 | 上升 | 疑问句/悬念，旋律上行 |
| 副歌 | Chorus | 核心表达 | 高潮 | 重复Hook，最强记忆点 |
| 桥段 | Bridge | 转折新视角 | 变化 | 与前文对比，新角度 |
| 说唱段 | Rap | 密集信息 | 驱动 | 快速节奏，大量歌词 |
| 尾声 | Outro | 收束留白 | 回落 | 渐弱重复或新结语 |
| 间奏 | Interlude | 音乐过渡 | 喘息 | 无词或极少词 |

## 3. 押韵系统

### 3.1 韵脚模式

| 模式 | 名称 | 结构 | 效果 |
|------|------|------|------|
| AABB | 连续韵 | 每两句押韵 | 稳定，朗朗上口 |
| ABAB | 交叉韵 | 隔行押韵 | 编织感，叙事 |
| ABBA | 环抱韵 | 首尾相押 | 回环，诗意 |
| ABCB | 偶数韵 | 只偶数行押 | 自然，不刻意 |
| AAAA | 全韵 | 每行都押 | 强调，rap常用 |
| 自由韵 | 无固定 | 不严格押韵 | 自由，口语化 |

## 4. 音韵学与旋律匹配

### 中文声调与旋律方向

| 声调 | 调值 | 自然旋律方向 | 建议 |
|------|------|------------|------|
| 一声阴平 | 55高平 | 高音区平稳 | 适合长音高音 |
| 二声阳平 | 35高升 | 上行 | 适合渐强旋律 |
| 三声上声 | 214降升 | 先降后升 | 适合装饰音 |
| 四声去声 | 51全降 | 下行 | 适合重音强拍 |

**核心规则**：旋律走向尽量与声调方向一致，尤其在关键词上。

## 5. Hook 设计技巧

### Hook 类型

| 类型 | 描述 | 示例 |
|------|------|------|
| 旋律Hook | 极简重复的旋律 | Let it go |
| 歌词Hook | 朗朗上口的短句 | We will rock you |
| 节奏Hook | 独特的节奏型 | Queen的跺脚+拍手 |
| 音色Hook | 标志性音色 | 合成器riff |
| 组合Hook | 旋律+歌词+节奏 | 最有效但最难 |

### Hook 写作技巧

| 技巧 | 说明 |
|------|------|
| 字数少 3-7字 | 短句更容易记忆 |
| 重复2-4次 | 副歌中反复出现 |
| 使用常用词 | 避免生僻词，口语化 |
| 元音开口 | 开口音更容易歌唱记忆 |
| 节奏独特 | 与主歌节奏形成对比 |

## 8. 歌词写作工作流

```
1. 确定主题和情感基调
2. 选择叙事模式（时间线/对话式/景到情...）
3. 确定结构（Verse-Chorus-Bridge...）
4. 先写Hook（最核心的一句）
5. 围绕Hook展开副歌
6. 写主歌（铺垫叙事导向副歌）
7. 写桥段（新角度转折）
8. 检查押韵和声调匹配
9. 添加AI标签（段落/表演/情感）
10. 精简打磨（删去赘词）
```

### 常见问题检查表

| 检查项 | 标准 |
|--------|------|
| 副歌是否有明确Hook | 3-7字可重复短句 |
| 主歌到副歌是否有能量提升 | 节奏加密/旋律上行 |
| 押韵是否一致 | 至少偶数行押韵 |
| 歌词是否口语化 | 大声读出来是否自然 |
| 每句字数是否适合旋律 | 每句5-15字为宜 |
| 是否有不必要的赘词 | 删去不影响含义的词 |
| 情感弧线是否清晰 | 有起伏不平淡 |
