美术开发原则 & UI 设计规范
风格定位：暗黑奇幻 x 二次元融合体 – “冰蓝哥特 · 少女幻想”

将欧美暗黑奇幻的史诗氛围感与日系二次元的精致角色表现力融合，
创造一种**“在黑暗世界中绽放的灵动之美”**的独特视觉体验。

通则：视觉纯净规范
禁止使用 Emoji / 表情符号
范围	规则
UI 文本	禁止在按钮、标签、提示文案中使用任何 Emoji 字符
图标系统	所有图标必须使用矢量绘制或精灵图，不得以 Emoji 替代
数据展示	货币、等级、状态等信息使用专用图标资源，不得用 Emoji 符号
通知/弹窗	提示类文案使用纯文字 + 自制图标，禁止 Emoji 装饰
设计稿标注	设计文档中不得以 Emoji 代替正式图标占位
理由: Emoji 风格不可控（各平台渲染差异大），与暗黑奇幻的精致美学格格不入，
且会破坏冷色调视觉的整体统一性。所有视觉元素必须经过美术管控。

第一章：美术开发原则
1.1 风格融合公式
最终风格 = 二次元角色表现（70%） + 暗黑奇幻氛围（30%）
         = 精致赛璐璐人物 + 冷色调光影环境
维度	二次元基因	暗黑奇幻基因	融合结果
角色	大眼、精致五官、柔和线条	复杂铠甲、暗黑装饰	穿着精致暗系装备的二次元角色
色彩	明快、高饱和度点缀	冷蓝主调、低饱和度大面积	冷色基底 + 局部高饱和点缀
光影	赛璐璐分层阴影	强烈明暗对比、自发光	赛璐璐基底 + 关键部位冷光辉
背景	干净、扁平化	雾气、山峦、废墟	虚化暗色场景 + 粒子光效
UI	圆角、轻量、半透明	金属质感、哥特纹样	磨砂玻璃 + 冰晶棱角装饰

A. 角色立绘 Prompt 框架
[必选项] 按顺序包含以下 6 个维度：

1. 主体描述  -- 角色身份、姿态、视线方向
2. 服装细节  -- 暗系哥特风 + 二次元轻盈感，冰蓝色金属铠甲元素
3. 色彩控制  -- 冷蓝主调(#0A0E1A~#6DC8F2)，发色限定冷色系
4. 光影指令  -- 赛璐璐分层 + 冰蓝自发光(武器/眼瞳/魔法纹)
5. 画面构成  -- 背景虚化暗色，前景粒子光效，景深分离
6. 品质锚点  -- 画质关键词锚定最终输出水准
角色 Prompt 模板:

核心理念：摄影镜头感（浅景深抓拍风格）
角色立绘不是"正面站立持武器"的证件照，而是被镜头捕捉到的自然瞬间。
像一位摄影师用中长焦镜头在角色身边抓拍到的一个动态剪影。

[镜头构图] -- 必选其一，决定整体印象：
  A. 仰拍近景  -- 镜头低于角色视线，角色俯视镜头，带来压迫感/神秘感
  B. 俯拍半身  -- 镜头高于角色，角色微微抬头或侧脸看向镜头，亲和/好奇
  C. 平视特写  -- 镜头平齐角色面部，脸部占画面 40%+，捕捉表情细节
  D. 侧身回眸  -- 身体侧向/背向镜头，头部转回看向观众，发丝飞扬
  E. 动态抓拍  -- 角色正在做某个动作的瞬间（翻书、伸懒腰、挥手、跳跃）

[姿态] -- 自然、有故事感，禁止僵硬站立：
  - 单手撑着脸颊，慵懒地看向镜头
  - 向前倾身探出，发丝垂落遮住半边脸
  - 坐在高处（窗台/废墟石块），双腿自然交叉悬空
  - 侧身靠在墙壁上，武器随手倚在身旁
  - 蹲下身摆弄什么东西，抬头被镜头捕捉到

[景深控制] -- 模拟真实摄影的虚化效果：
  - 前景虚化: 飘散的花瓣/光粒子/发丝梢部（高斯模糊 8-16px）
  - 焦点层: 角色面部和上半身清晰锐利
  - 背景虚化: 哥特建筑/环境大幅虚化（散景光斑），仅保留轮廓暗示

[完整 Prompt 结构]:

紫色瞳孔中有星辰般的冷光点。
[武器自然地放置在身旁/背后/手边，而非刻意持握展示]。

C. 背景图 Prompt 框架
[场景背景] Prompt 结构：

1. 场景类型  -- 哥特城堡/冰封荒原/暗夜森林/魔法废墟
2. 氛围关键词 -- 幽暗、神秘、寒冷、史诗感
3. 色调限定  -- 深蓝-冰蓝渐变色调，亮度 15%-25%
4. 景深层次  -- 前景(虚化碎片) + 中景(建筑) + 远景(天空/雾)

6. 模糊程度  -- 整体高斯模糊 8-16px（不抢角色焦点）
D. 一致性控制规则
规则	说明
色温锁定	所有产出色温 <= 5000K（冷色），禁止暖黄/橙色大面积出现
线条风格	外轮廓 2-3px 粗线 + 内部 1px 细线，线色随区域变化（非纯黑）
瞳孔规范	统一使用紫色/冰蓝色大瞳孔，内含 2-3 个高光点
发色限定	仅限：银白 / 冰蓝 / 暗紫 / 月光灰 / 深蓝黑
自发光点	每个角色至少 1-2 处自发光元素（武器/眼瞳/纹路）
铠甲质感	暗蓝金属底色 + 冰蓝色边缘高光 + 微量腐蚀纹理
背景亮度	整体亮度 15%-25%，绝不超过 30%
产出校验	每张图必须通过以下检查才可使用
E. 产出校验清单
每张 AI 生成图片交付前，必须逐项确认：

[ ] 主色调为冷蓝色系，无大面积暖色
[ ] 角色为二次元赛璐璐风格，非写实/厚涂
[ ] 角色瞳孔含冷色光点
[ ] 至少有 1 处自发光元素
[ ] 背景亮度未超过 30%
[ ] 无 Emoji 或文字水印
[ ] 线条清晰，无明显 AI 伪影（多余手指/畸变）
[ ] 色彩与核心色板偏差 < 15%（通过取色验证）
[ ] 输出分辨率满足最低要求（立绘 >= 2048px，UI >= 512px）
1.4 角色立绘规范
核心理念：摄影艺术感（中长焦浅景深 · 自然抓拍风格）
本项目立绘的辨识度在于它不是"角色展示图"，而是**“角色生活照”**。
镜头仿佛是一位跟在角色身边的摄影师，在某个瞬间按下快门。
角色在画面中不是"摆 Pose 给你看"，而是"被你看到了"。

镜头语言规范（核心差异点）
维度	传统立绘	摄影抓拍式立绘（本项目采用）
构图	全身正面居中	半身/近景，偏向一侧，留白不对称
视角	平视正面	仰拍/俯拍/侧面回眸，有镜头倾斜感
姿态	站立持武器	自然动态（倚靠/前倾/侧坐/抬手遮光）
武器	居中展示	自然搁置（靠在肩上/拖在地上/放在身旁）
景深	无/弱	强浅景深，前景虚化光斑+背景散景
表情	标准微笑/严肃	有情绪的瞬间（好奇/慵懒/微怔/戏谑）
发丝	静态造型	被风/动作带起，遮挡部分面庞增加层次
光源	均匀	侧光/逆光/环境反射光，有明确光源方向
推荐构图方案（按优先级）
构图 A "俯瞰凝视" (优先级最高，本项目经典构图):
  - 镜头位于角色正下方或斜下方
  - 角色向前倾身/俯下身，从上方看向镜头
  - 发丝因重力自然垂落，遮住部分面庞
  - 适合表现：好奇、关切、亲近感
  - 参考: 角色俯身探出，发丝垂落，浅景深虚化背景

构图 B "侧身近景":
  - 角色身体朝向画面左/右侧，头部微转面向镜头
  - 半身入画，肩部以上清晰，手臂/武器可截出画面边缘
  - 浅景深，背景大幅虚化
  - 适合表现：沉静、从容、气场

构图 C "动态抓拍":
  - 角色正在做某个动作的瞬间被捕捉
  - 翻阅魔法书/调整铠甲/回头看到什么/伸手触碰光粒子
  - 有运动模糊暗示（发梢/衣摆边缘微糊）
  - 适合表现：活力、故事感

构图 D "坐姿仰望":
  - 角色坐在高处（废墟石块/哥特窗台/冰晶throne）
  - 镜头从较低位置仰拍，双腿自然悬空或交叉
  - 武器随手倚在身旁
  - 适合表现：王者气度、悠然
绘制风格
项目	规范
线稿	外轮廓粗线（2-3px）+ 内部细线（1px），线条颜色随区域变化（非纯黑）
上色	赛璐璐为主，2-3 层阴影分层，关键区域（眼睛/武器/魔法纹）加柔光效果
眼睛	二次元大眼风格，瞳孔加入冰蓝色光点，体现暗黑奇幻的超自然感
服装	哥特/暗系设计，但保留二次元的轻盈感；金属铠甲部分用冰蓝高光表现
发色	偏冷色调（银白/冰蓝/暗紫/月光灰），发尾可加冷色渐变
发丝	必须有动态感：被风吹起/因动作飘散/因重力垂落，遮挡部分面庞增加层次
光效	角色身上至少 1-2 处自发光元素（武器光芒/魔法阵/眼瞳光）
光源	必须有明确方向光：侧光或逆光为主，形成冷色轮廓光描边角色边缘
景深	必须有浅景深效果：前景散焦粒子 + 背景虚化散景光斑

1.5 背景美术规范
首页背景分层结构（摄影景深版）
Layer 6 (最上层): 前景虚化层 -- 模拟浅景深的"镜头前方"散焦元素
                  散焦光斑 x8-12: 大尺寸(r=30-80px) 低透明度圆形, 高斯模糊
                  飘散冰晶碎片: 不规则形状, alpha 10-25%, 模糊 12-20px
                  目的: 制造"透过镜头看角色"的沉浸感

Layer 5: 粒子效果 -- 冰蓝光粒子 x60，三层光晕渲染
                  外层: 半径 size*5, alpha 8%, 色值 #3A7BD5
                  中层: 半径 size*2.5, alpha 25%, 色值 #6DC8F2
                  核心: 半径 size, alpha 100%, 色值 #C8EBFF
                  部分近处粒子做散焦处理(融入 Layer 6)

Layer 4: 角色立绘 -- 摄影级构图（半身近景/俯瞰凝视/侧身回眸）
                  带冰蓝光晕氛围（半径300px径向渐变，以面部为圆心）
                  底部 400px 渐隐融合入背景
                  角色为画面**焦点层**，清晰锐利

Layer 3: 雾气层   -- 三层动态雾气（缓慢水平漂移），处于焦外
                  上部雾: Y=300, 高350px, alpha 40
                  中部雾: Y=700, 高250px, alpha 30
                  底部雾: Y=1200, 高500px, alpha 200（融合角色下半身）

Layer 2: 建筑剪影 -- 哥特建筑矢量剪影，**背景散景效果**
                  低透明度(alpha 35), 微高斯模糊(4-8px) 模拟焦外
                  左侧尖塔 + 右侧城堡 + 中间远景小城堡(alpha 21)

Layer 1 (最底层): 深蓝渐变基底 -- #080C16 到 #0E1424 纵向渐变
景深分层原则：Layer 6(前景虚) → Layer 4(角色清晰) → Layer 2-1(背景虚)
形成 “近处散焦 — 角色锐利 — 远处散景” 的三段景深，还原真实摄影效果。

氛围要求
背景整体亮度控制在 15%-25%，确保不抢角色视觉焦点
必须有至少一层雾气/光晕层，增加空间纵深
建筑剪影使用矢量路径绘制，保持低透明度(alpha <= 35)，仅提供轮廓暗示
雾气层各自独立漂移，周期不同，增加自然感
允许微动效：雾气缓慢流动、光点轻微闪烁、角色呼吸浮动
前景散焦层必须存在：至少 8 个大尺寸低透明度散焦光斑，增加镜头纵深感
背景应有散景暗示：建筑剪影边缘微模糊，模拟焦外效果
