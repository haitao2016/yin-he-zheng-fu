# 风格迁移 Prompt 模板库

> art-style-transfer Skill 的完整 prompt 模板参考。
> 覆盖 15 种内置风格 × 5 大素材品类的组合模板。

---

## §1 Prompt 构建公式

### 通用公式

```
[操作指令] + [风格维度描述] + [品类约束] + [技术要求]
```

**四个组成部分**：

| 部分 | 说明 | 示例 |
|------|------|------|
| 操作指令 | 迁移动作 | "将此图转换为…风格" |
| 风格维度描述 | 6 维度特征 | "厚重笔触、暖色调、柔和光影" |
| 品类约束 | 素材类型限制 | "保持纹理可平铺性" |
| 技术要求 | 输出规格 | "保留透明通道" |

### 变量占位符

模板中使用以下占位符，实际使用时替换为具体值：

| 占位符 | 含义 | 示例值 |
|--------|------|--------|
| `{style_name}` | 风格名称 | 印象派油画 |
| `{asset_type}` | 素材类型 | 地面纹理 |
| `{preserve_level}` | 内容保留度 | high / medium / low |
| `{intensity}` | 风格强度 | light / medium / strong |
| `{resolution}` | 输出分辨率 | 512x512 |

## §5 风格混合 Prompt

当需要混合两种风格时，使用以下模板：

### 双风格混合

```
将此{asset_type}转换为{style_A}与{style_B}的混合风格。
主基调采用{style_A}的{维度1}和{维度2}，
融入{style_B}的{维度3}元素。
{品类约束}。{技术要求}。
```

**示例 — 赛博朋克 × 浮世绘**：
```
将此角色立绘转换为赛博朋克与浮世绘的混合风格。
主基调采用赛博朋克的霓虹配色和科技线条，
融入浮世绘的平涂色块和精细描线元素。
保持角色轮廓清晰可辨。
```

**示例 — 水墨 × 低多边形**：
```
将此场景背景转换为中国水墨与低多边形的混合风格。
主基调采用水墨的黑灰留白和渲染效果，
山石树木使用低多边形的棱面几何简化。
保持场景层次分明。
```

## §6 内容保留度修饰语

根据 SKILL.md 中的三级内容保留度，在 prompt 末尾追加对应修饰语：

### high（高保留 — 纹理贴图常用）

```
严格保持原始图像的结构、布局和细节不变，
仅改变色彩风格和表面质感。所有元素的位置、
大小、形状必须与原图一致。
```

### medium（中保留 — 角色/UI 常用）

```
保持原始图像的主要结构和关键特征，
允许适度的风格化变形。核心识别元素必须保留。
```

### low（低保留 — 场景背景常用）

```
以原始图像为灵感参考，允许大幅度的风格化重新诠释。
保持整体氛围和主题，但构图和细节可以自由发挥。
```

## §7 技术约束速查

不同素材品类的固定技术约束，附加在 prompt 末尾：

| 品类 | 技术约束 |
|------|---------|
| 纹理贴图 | "保持纹理可平铺特性，确保边缘无缝衔接" |
| UI 界面 | "保持文字可读性，交互区域清晰明确" |
| 角色立绘 | "保持角色整体轮廓识别度，面部特征可辨" |
| 场景背景 | "保持场景层次分明，前中后景区分清晰" |
| 图标道具 | "保持图标在小尺寸下的可识别性" |

### 透明背景追加

当素材需要透明背景时，追加：
```
生成透明背景版本，主体边缘干净利落。
```

### 分辨率追加

当有特定分辨率要求时，追加：
```
输出分辨率为{width}x{height}像素。
```
