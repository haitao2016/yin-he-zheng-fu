# 刷刷乐 JRPG 设计参考

> 代表作：Disgaea 系列、魔界战记、最终幻想 Tactics、Octopath Traveler、勇者斗恶龙怪物篇、Pokemon、七龙珠Z：卡卡罗特
>
> 核心体验：**玩家为了"再强一点"而持续游戏**。系统深度和数值层叠是核心乐趣——理解系统 = 指数级效率提升，形成"研究→执行→收益→更深研究"的正循环。

- 转生、觉醒、素材阶梯、装备合成
- 排行榜、竞技场、日课、赛季
- 角色收集、角色池、角色培养

## 预设总纲（取 3 条）

| # | 总纲 | 一句话 |
|---|------|--------|
| L1 | **奖励知识，不奖励时间** | 乘算倍率让"懂得多"远比"磨得久"高效；信息是最强货币 |
| L2 | **经济有入口也有出口** | 来源多样，去向也多样；零产出内容防通胀，隐性成本防无限积累 |
| L3 | **丰而不强制** | 70-80% 内容可跳过；强制的只有主线最小路径 |

搭配通用总纲建议：G3（无孤立系统）、G4（同一内容多层体验）

## 模块特化指南

### 模块 1: 内容分层（刷刷特化）

**结构：表里双世界**

| 维度 | 表游戏 | 里游戏 |
|------|--------|--------|
| 时长 | 50h（推荐） | ≥80h（推荐） |
| 内容密度 | 线性递增 | 指数递增 |
| 系统开放度 | 渐进解锁 | 全部开放 |
| 难度 | 引导式 | 玩家自定义 |

**可选层占比 70-80%**：
- 隐藏区域、隐藏 Boss、隐藏角色
- 高难迷宫、制约挑战
- 收集系统、成就系统
- 全部可跳过，主线仍完整

### 模块 2: 经济系统（刷刷特化）★核心★

**完整采用 [base-framework](base-framework.md) 的经济模块**，并强化以下子系统：

#### 倍率乘算（必须）

```
最终收益 = 基础值 × 评价倍率 × 道具倍率 × 系统倍率 × 角色倍率

├─ 评价倍率：战斗评价等级       ×1.0 ~ ×1.5
├─ 道具倍率：增益消耗品         ×1.0 ~ ×2.0
├─ 系统倍率：被动技能/设施加成   ×1.0 ~ ×2.5
└─ 角色倍率：角色特性/装备       ×1.0 ~ ×2.0

全叠加理论最大 ≈ ×15
```

理解系统的玩家收益是不理解的 **15 倍**。这就是"奖励知识"的兑现。

#### 素材阶梯（必须）

| 层级 | 获取途径 | 瓶颈 | 流转 |
|------|---------|------|------|
| 1 废料 | 低级区域战斗掉落 | 否 | → 合成层级 2 |
| 2 下级 | 中级区域战斗掉落 | 否 | → 合成层级 3 |
| **3 中级** | **购买 + 分解** | **是** | → 合成层级 4 |
| 4 上级 | Boss 掉落 / 栽培 | 否 | → 合成层级 5 |
| 5 神级 | 限定来源 | 否 | → 最终装备 |

**瓶颈设计是关键**：第 3 层是金钱出口——店购装备→分解→素材→合成。

#### 放置型设施（推荐）

| 设施 | 投入 | 产出 |
|------|------|------|
| 生产设施 A | 角色 + 时间 | 素材 + 被动经验 30% |
| 生产设施 B | 角色 + 时间 | 素材 + 被动经验 30% |
| 收集装置 | 放置到地图 + 时间 | 地图掉落品（兼作传送点） |

#### 城镇投资（推荐）

阶梯式捐赠 → 城镇等级 → 解锁店铺/设施/NPC。超高额目标（百万级）作为终极出口。

### 模块 3: 战斗系统（刷刷特化）

**属性体系必须完整**：

- 6-10 属性环形相克
- 含 1 个中性安全阀属性（穿透所有耐性/无弱点）
- 相克倍率：克制 ×1.5 / 被克 ×0.5

**Boss 规则修改器库**：

| ID | 规则 | 迫使 |
|----|------|------|
| 物理反击 | 物理攻击必被反击 | 切换魔法 |
| 魔法反击 | 魔法攻击必被反击 | 切换物理 |
| 异常免疫 | Debuff 无效 | 纯伤害突破 |
| 伤害稳定 | 单次伤害有上限 | 多段攻击 |

Boss = 谜题，不是数值墙。

**战斗评价系统**：
| 评级 | 条件 | 倍率 |
|------|------|------|
| S | 无伤+速杀+全弱点 | ×1.5 |
| A | 速杀或全弱点 | ×1.3 |
| B | 正常通过 | ×1.0 |
| C | 困难通过 | ×0.8 |

### 模块 4: 角色与成长（刷刷特化）★核心★

**大角色池 + 多维成长**：

#### 角色池结构

| 类型 | 占比 | 特征 |
|------|------|------|
| 常规角色 | 85-90% | 各有独占能力 |
| 跳板角色 | 1-3 个 | 经验极低，快速到高段位，继承入队等级 |
| 补偿角色 | 1-2 个 | 后期加入，通用性强，兜底 |

#### 转生系统（必须）

| 转生次 | 解锁 | 性质 |
|--------|------|------|
| 1 | 被动增强 | 被动质变 |
| 2 | 高阶主动技能 | 主动质变 |
| 3-5 | 逐层解锁 | 每次须有质变 |

- 转生后回 Lv.1，保留装备+技能+内容
- 能力点一本化：转生次数 × 2

#### 相性系统

**只有好相性奖励，无坏相性惩罚**：
- 同组编队 → 联携技/属性+10%
- 非同组 → 正常数值（不扣减）

#### 零板凳原则

所有角色至少占据一个分配位（战斗队/预备队/生产设施），持续贡献。

### 模块 5: 装备与合成（刷刷特化）★核心★

**深度合成链**：

- 统一造价：所有角色最终装备造价一致（公平）
- 统一素材链层级：3 层
- 统一制作步骤：3 步

**异常免疫拼图**：
- 全免疫 = 饰品槽全部防御 → 放弃攻击饰品
- 不存在无代价完美防御

**阶段策略变化**：
| 阶段 | 武器策略 | 防具策略 |
|------|---------|---------|
| 序盘 | 直接购买 | 直接购买 |
| 中盘 | 合成武器 | 店购防具 |
| 终盘 | 全最强合成 | 全最强合成 |

### 模块 6: 探索与地图（刷刷特化）

**昼夜系统服务掉落多样性**：

| 参数 | 推荐值 |
|------|--------|
| 白天时长 | 游戏内 12h（现实 10min） |
| 夜间时长 | 游戏内 10h（现实 8min） |
| 特殊时段 | 每 7 日 1 次，持续 2h |
| 夜间报酬 | 永久属性道具（非消耗品） |

**密语收集**：4 字密语 30-50 个，环境文本载体，特定 NPC 兑换。

### 模块 7: 任务与联动（刷刷特化）

**跨系统前置是关键**：

```
| 解锁内容 | 条件 A（系统 X） | 条件 B（系统 Y） | 关系 |
|---------|----------------|----------------|------|
| 经营设施 | 城镇等级 ≥2 | 角色技能 ≥3 | AND |
| 隐藏角色 | 胜利 ≥N 次 | 收集 ≥N% | OR |
```

**贯穿元素**：至少 1 个元素贯穿 ≥3 系统。

**行为累积成就**：静默追踪自然行为，达阈值惊喜解锁。

### 模块 8: 玩家心理（刷刷特化）

**安全网五层全启用**（一条不少）：

1. 金钱保护（战败损失 0%）
2. 无 Game Over（全灭→据点复活）
3. 即时复归（复活后可传送回倒下位置）
4. 手动撤退（战中可放弃，无惩罚）
5. 败北报酬（全灭获耐性装备）

**聚焦选择器**：金钱×2 / 经验×2 / 掉落×2，互斥，自由切换。

**容许技巧**：挂机刷、倒卖利润、卡时间 → 全部保留不封堵。

### 模块 9: 通关后留存（刷刷特化）★核心★

#### 制约迷宫

| 参数 | 推荐值 |
|------|--------|
| 层数 | 50-100 |
| 检查点 | 每 10 层 |
| 剥夺规则 | 禁道具/禁传送/禁金钱（可叠加） |

#### Boss 链

通关后 Boss 散布整个世界，链式击破，覆盖多区域。每次击破后 +5-10% 全属性。

#### 日课循环

| 日课 | 机制 | 报酬 |
|------|------|------|
| 随机商品 | 每日刷新 | 稀有道具 |
| 每日挑战 | 随机制约 Boss 战 | 限定素材 |

#### 挑战即报酬

败北零损失 + 败北有收获 = 无理由不尝试。

## 数值曲线参考

### 经验曲线

```
所需经验 = base × (level ^ exponent)
推荐: base = 100, exponent = 1.5

Lv.1→2: 100
Lv.10→11: 3,162
Lv.50→51: 35,355
Lv.99→100: 98,506
```

### 伤害公式

```
伤害 = (ATK × skill_mult - DEF × 0.5) × elemental_mult × random(0.95, 1.05)

elemental_mult:
  克制 = 1.5
  中性 = 1.0
  被克 = 0.5
```

### 通胀控制

| 阶段 | 推荐金钱范围 | 主要出口 |
|------|------------|---------|
| 序盘 | 0-10,000 | 装备购买 |
| 中盘 | 10,000-100,000 | 合成+城镇投资 |
| 终盘 | 100,000-1,000,000 | 统一造价+超高额投资 |
| 里游戏 | 1,000,000+ | 制约迷宫消耗+日课 |

## 推荐音频风格

刷刷乐型音乐以"长时间不厌倦"和"正反馈强化"为核心，节奏驱动。

| 场景 | 风格关键词 | AI 作曲提示词参考 |
|------|-----------|-----------------|
| 主菜单 | epic, triumphant, bold | `epic JRPG menu theme, triumphant brass, bold and inviting, orchestral` |
| 城镇/据点 | upbeat, cozy, looping | `cozy town theme, upbeat acoustic, gentle loop, relaxing but not sleepy` |
| 表游戏探索 | light orchestral, adventure | `light adventure exploration, orchestral, cheerful, moderate tempo` |
| 里游戏探索 | darker, intense, synth + orchestral | `dark post-game exploration, synth and orchestral hybrid, intense atmosphere` |
| 普通战斗 | fast, rhythmic, rock + electronic | `fast-paced battle, rock guitar and electronic beat, energetic, loopable` |
| Boss 战 | heavy, phase-shifting, choir | `heavy boss battle, shifting phases, dramatic choir, intense percussion, building` |
| 制约迷宫 | tense, minimalist, ambient | `tense dungeon crawl, minimalist ambient, subtle percussion, oppressive` |
| 战斗评价S | fanfare, rewarding, short | `triumphant S-rank fanfare, short brass celebration, rewarding jingle` |
| 合成/强化成功 | satisfying, sparkle, short | `satisfying upgrade success, sparkle sound, short celebratory chime` |
| 转生演出 | grand, transformation, crescendo | `grand transformation theme, crescendo, ethereal choir, rebirth, powerful` |
| 日课/挑战 | uptempo, competitive | `uptempo challenge theme, competitive, timer-like urgency, exciting` |

**音乐设计原则**：
- 耐听优先：主要刷图音乐使用中等节奏、避免极端高音或刺耳音色，循环点自然
- 正反馈音效链：评价升级（B→A→S）配套递进式音效，倍率激活有专属短乐句
- 里游戏变奏：里游戏使用表游戏主题的暗调变奏（加小调/加合成器层），体现进阶感
- 沉默时刻：制约迷宫高层可减至纯环境音+心跳低频，强化压迫感
