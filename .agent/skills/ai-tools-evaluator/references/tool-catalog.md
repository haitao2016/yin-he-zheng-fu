## 1. 3D 模型生成工具

### 1.1 Meshy

- **定位**：通用 3D 模型生成，速度与质量平衡
- **输入**：文本描述 / 图片（正面视图）
- **输出格式**：GLB、FBX、OBJ、USDZ
- **质量评级**：★★★★☆（4/5）
- **生成速度**：2-5 分钟
- **定价**：
  - 免费：20 次/月
  - Pro：$20/月（200 次/月）
  - Max：$48/月（500 次/月）
- **优势**：速度快、贴图质量好、拓扑结构合理
- **劣势**：复杂角色细节有限、免费额度少
- **UrhoX 集成**：GLB → import-glb skill → MDL

### 1.2 Rodin (Deemos)

- **定位**：高精度 3D 模型生成
- **输入**：文本 / 图片 / 多视角图片
- **输出格式**：GLB、FBX
- **质量评级**：★★★★★（5/5）
- **生成速度**：5-15 分钟
- **定价**：按次付费，约 $0.5-2/个
- **优势**：几何细节极其丰富、表面精度高
- **劣势**：速度较慢、成本较高
- **UrhoX 集成**：GLB → import-glb skill → MDL

### 1.3 Luma Genie / Dream Machine

- **定位**：快速 3D 原型制作
- **输入**：文本 / 图片 / 视频
- **输出格式**：GLB、OBJ
- **质量评级**：★★★☆☆（3/5）
- **生成速度**：1-3 分钟
- **定价**：免费试用，Pro $30/月
- **优势**：速度极快、适合快速原型
- **劣势**：细节不如 Meshy/Rodin、拓扑质量一般
- **UrhoX 集成**：GLB → import-glb skill → MDL

### 1.4 Scenario.com

- **定位**：风格一致的 2D/3D 游戏素材批量生成
- **输入**：文本 + 自定义 LoRA 风格模型
- **输出格式**：PNG（2D）、GLB（3D）
- **质量评级**：★★★★☆（4/5）
- **生成速度**：中等
- **定价**：免费 100 次/月，Pro $30/月
- **优势**：可训练自定义风格、批量一致性好
- **劣势**：3D 功能相对较新
- **UrhoX 集成**：PNG 直接用 / GLB → import-glb

### 1.5 Ready Player Me

- **定位**：跨平台游戏角色 Avatar
- **输入**：照片 / 自定义选项
- **输出格式**：GLB
- **质量评级**：★★★☆☆（3/5）
- **生成速度**：< 1 分钟
- **定价**：免费
- **优势**：免费、速度快、跨平台一致
- **劣势**：风格固定（卡通风）、自定义有限
- **UrhoX 集成**：GLB → import-glb skill → MDL

### 1.6 UrhoX 内置 (create_3d_model_task)

- **定位**：UrhoX 原生 3D 模型生成
- **输入**：文本 / 正面图片 / 多视角图片
- **输出格式**：MDL（原生）
- **质量评级**：★★★★☆（4/5）
- **生成速度**：1-5 分钟
- **定价**：免费
- **优势**：原生格式无需转换、支持自动绑骨（rig=true）、免费
- **劣势**：风格控制不如专业工具
- **UrhoX 集成**：✅ 原生支持，无需转换

## 2. PBR 纹理/贴图工具

### 2.1 Adobe Substance 3D Sampler

- **定位**：行业标准 PBR 材质生成
- **输入**：照片 → AI 分析生成完整 PBR 套件
- **输出**：Albedo、Normal、Roughness、Metallic、Height（PNG/EXR）
- **质量评级**：★★★★★（5/5）
- **定价**：$49.99/月（Adobe 订阅）
- **优势**：专业级品质、完整 PBR 工作流
- **劣势**：价格高、学习曲线陡
- **UrhoX 集成**：导出 PNG → PBR 材质 XML 引用

### 2.2 DreamTextures

- **定位**：Blender 内置纹理生成插件
- **输入**：文本描述
- **输出**：PNG（在 Blender 内）
- **质量评级**：★★★★☆（4/5）
- **定价**：免费（开源）
- **优势**：免费、Blender 工作流内闭环
- **劣势**：需要安装 Blender
- **UrhoX 集成**：导出 PNG → assets/Textures/

### 2.3 Polycam

- **定位**：手机扫描生成 3D 资产和贴图
- **输入**：手机 LiDAR / 多张照片
- **输出**：PNG、OBJ、GLB
- **质量评级**：★★★☆☆（3/5）
- **定价**：免费基础版，Pro $8/月
- **优势**：扫描真实物体、低价
- **劣势**：质量依赖拍摄条件
- **UrhoX 集成**：纹理 PNG → assets/Textures/

## 3. 2D 图像生成工具

### 3.1 Midjourney v7 / Niji 7

- **定位**：最高画质的 AI 图像生成
- **输入**：文本描述（Discord / Web）
- **输出**：PNG、JPG
- **质量评级**：★★★★★（5/5）
- **定价**：$10/月起（无免费额度）
- **优势**：画质最佳、写实/二次元皆强
- **劣势**：无免费额度、Discord 工作流
- **UrhoX 集成**：PNG → assets/Textures/

### 3.2 Recraft V3

- **定位**：UI 设计与矢量图生成
- **输入**：文本描述
- **输出**：PNG、SVG（矢量）
- **质量评级**：★★★★★（5/5）
- **定价**：免费 50 次/天，Pro $25/月
- **优势**：SVG 矢量输出、UI/图标专精
- **劣势**：通用场景不如 Midjourney
- **UrhoX 集成**：PNG → assets/Textures/（SVG 需先转 PNG）

### 3.3 Leonardo AI

- **定位**：游戏资源专属图像生成
- **输入**：文本描述 + 可选参考图
- **输出**：PNG（支持透明背景）
- **质量评级**：★★★★☆（4/5）
- **定价**：免费 150 token/天，$12/月
- **优势**：透明背景、3D 纹理模式、私有模型训练
- **劣势**：免费额度有限
- **UrhoX 集成**：PNG → assets/Textures/

### 3.4 Stable Diffusion (WebUI / ComfyUI)

- **定位**：最强控制力的开源图像生成
- **输入**：文本 + ControlNet（姿势/构图控制）
- **输出**：PNG
- **质量评级**：★★★★★（5/5）
- **定价**：免费（开源，需本地 GPU）
- **优势**：完全控制、ControlNet 精确控制、无限生成
- **劣势**：需要显卡、学习曲线高
- **UrhoX 集成**：PNG → assets/Textures/

### 3.5 Flux.1

- **定位**：开源界最强文生图模型
- **输入**：文本描述
- **输出**：PNG
- **质量评级**：★★★★★（5/5）
- **定价**：开源免费 / API 付费
- **优势**：文字渲染极强、画面接近 Midjourney
- **劣势**：本地部署需大显存
- **UrhoX 集成**：PNG → assets/Textures/

### 3.6 UrhoX 内置 (generate_image)

- **定位**：引擎原生图像生成
- **输入**：中文描述 + 可选参考图（最多 14 张）
- **输出**：PNG（支持透明背景、自定义尺寸）
- **质量评级**：★★★★☆（4/5）
- **定价**：免费
- **优势**：免费、集成度最高、支持批量、支持编辑
- **劣势**：复杂场景可能不如专业工具
- **UrhoX 集成**：✅ 原生支持
