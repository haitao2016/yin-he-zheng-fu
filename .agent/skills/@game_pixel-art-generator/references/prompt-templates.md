# 像素风提示词模板

## 通用前缀

所有像素风素材 prompt 必须以风格前缀开头，确保生成一致的像素风格。

**基础前缀**：`像素风格,pixel art,复古游戏风格,清晰像素边缘,无抗锯齿`

**透明背景前缀**（角色/道具）：`像素风格,pixel art,复古游戏风格,清晰像素边缘,无抗锯齿,透明背景`

---

## 1. 角色素材

### 1.1 静态角色

```
像素风格,pixel art,复古游戏风格,清晰像素边缘,无抗锯齿,透明背景,{角色描述},{朝向},全身像
```

**角色描述示例**：
- 勇士：`手持长剑的骑士,银色铠甲,蓝色披风`
- 法师：`紫色长袍法师,手持法杖,尖顶帽`
- 盗贼：`黑色斗篷盗贼,手持匕首,面罩`
- 弓箭手：`绿衣精灵弓箭手,长弓,箭袋`
- 怪物：`绿色史莱姆,圆形果冻状,两只大眼睛`

**朝向**：`正面` / `侧面朝右` / `侧面朝左` / `45度侧面`

### 1.2 动画帧 - 行走

生成 4 帧行走动画（侧面视角），每帧独立生成：

| 帧 | prompt 后缀 |
|----|------------|
| 帧1（站立） | `侧面朝右,站立姿势,双脚并拢` |
| 帧2（迈左腿） | `侧面朝右,行走姿势,左腿向前迈出,右腿在后` |
| 帧3（中间） | `侧面朝右,行走姿势,双腿交叉中间过渡` |
| 帧4（迈右腿） | `侧面朝右,行走姿势,右腿向前迈出,左腿在后` |

### 1.3 动画帧 - 攻击

生成 3 帧攻击动画：

| 帧 | prompt 后缀 |
|----|------------|
| 帧1（蓄力） | `侧面朝右,攻击蓄力姿势,武器举起后方` |
| 帧2（挥击） | `侧面朝右,攻击挥击中,武器向前劈下` |
| 帧3（收招） | `侧面朝右,攻击完成姿势,武器前伸到位` |

### 1.4 动画帧 - 待机

生成 2 帧待机呼吸动画：

| 帧 | prompt 后缀 |
|----|------------|
| 帧1 | `侧面朝右,待机站立,正常姿势` |
| 帧2 | `侧面朝右,待机站立,微微上下浮动,呼吸感` |

## 5. 提示词优化技巧

### 增强像素感的附加词

- 低分辨率感：`低像素数,8-bit风格`
- 高对比度：`高对比色彩,饱和色调`
- 特定时代风格：`FC/NES风格` / `SFC/SNES风格` / `GBA风格`
- 限制色板：`16色调色板` / `32色调色板`

### 避免的描述

- 不要用 `照片级真实` / `写实` / `3D渲染`
- 不要用 `模糊` / `柔和过渡` / `渐变`
- 不要用 `高分辨率细节` / `精细毛发`
