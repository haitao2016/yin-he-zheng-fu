## 注意事项

1. **generate_image 工具**的 AI 生成分辨率远大于目标尺寸，最终会缩放到 target_size。小尺寸（16x16）缩放后像素边缘可能模糊，建议 prompt 强调 `低像素数,粗大像素块,清晰方形像素`。
2. **batch_generate_images** 适合批量生成同一套素材（如一组动画帧、一套道具图标），所有图片并行生成效率更高。
3. 角色和道具通常需要**透明背景**（`transparent: true`），场景背景不需要。
4. 动画帧应使用**相同的 seed** 以确保角色外观一致性。
