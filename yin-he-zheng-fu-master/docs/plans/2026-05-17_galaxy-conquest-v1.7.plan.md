## 1. 项目概述

**银河征服** V1.7 在 V1.6 功能完整版基础上，聚焦三大方向：
- **内容缺口补全**：填补科技树 Tier4 路线断层、基地核心 Lv8-10 空解锁、探索任务模板匮乏等明确数据漏洞
- **游戏深度强化**：无尽模式 Roguelike 化轮间选卡、战斗战中目标指令、隐藏成就链式解锁
- **多人体验提升**：快捷信号通信系统、EndGame 战绩雷达图对比

**技术栈**：UrhoX 引擎 + raw NanoVG 渲染 + 多人网络（Client/Server 架构）

## 5. 功能详细说明

### P1-1 科技树双路线 Tier4 补全

**背景**：当前 Tier4 的 `QUANTUM_CORE` 和 `PHASE_DRIVE` 均要求 `WARP_DRIVE` 作为前置，导致选择 ADVANCED_WEAPONS（攻击路线）或 DEFENSE_MATRIX（防御路线）的玩家**永远无法到达 Tier4**，科技上限人为偏低。

**新增 2 项 Tier4 科技**：

| 科技 | 路线前置 | 效果 | 成本 |
|------|---------|------|------|
| `NOVA_CANNON` 新星炮 | ADVANCED_WEAPONS | 战舰 AOE 半径+80%，每波战斗开始额外获得 1 次"星陨打击"主动技能 | nuclear=500, metal=800, esource=400, time=85s |
| `FORTRESS_PROTOCOL` 要塞协议 | DEFENSE_MATRIX | 基地最大护盾×2.2，战斗中护盾每10s自动恢复 5% | nuclear=400, metal=600, esource=600, time=80s |

**TechPanel UI 改动**：
- Tier4 行显示 3 个格子（QUANTUM_CORE / NOVA_CANNON / FORTRESS_PROTOCOL），依所选 Tier3 路线高亮可研/锁定
- 已锁定格子显示为深灰并标注"路线锁定（需 ADVANCED_WEAPONS）"字样

**Systems.lua 效果实现**：
- `NOVA_CANNON.bonus = { aoeRadiusMult=1.8, battleStartSkillCharge=1 }` — BattleScene 读取 aoeRadiusMult 放大 AOE 半径，battleStartSkillCharge 在波次开始时触发
- `FORTRESS_PROTOCOL.bonus = { shieldMaxMult=2.2, shieldRegenPct=0.05 }` — BattleScene 每10s 检查 shieldRegenPct 并回血

### P1-3 探索任务模板扩展

**背景**：`EXPLORER_TASK_TEMPLATES` 仅 6 条，普通局10分钟内全部见过；无系列任务链，探索失去惊喜感。

**扩展至 20 条模板**（保留原6条，新增14条）：

新增模板示例（含系列任务链标记）：
```lua
{ label="调查时空异常",   minDur=45, maxDur=65, icon="🌀",
  rewards={{res="nuclear",amt=200},{res="credits",amt=100}}, expGain=130,
  chainNext="RIFT_FOLLOWUP" },  -- 有50%概率触发后续链任务
{ label="破解古代密码",   minDur=60, maxDur=90, icon="🏺",
  rewards={{res="credits",amt=400}}, expGain=200, isChain=true,
  chainFrom="RIFT_FOLLOWUP", rareReward=true },
```

**系列任务链机制**：
- 带 `chainNext` 字段的任务完成后，30s 内以 50% 概率触发对应链式后续任务
- 链式后续任务在通知栏展示特殊图标（🔗）和来源说明
- 链式任务奖励翻倍，expGain 额外+50%

**20 条模板分类**：

### P2-1 无尽模式轮间选卡

**背景**：无尽模式 `startEndlessNextRound()` 在歼灭所有海盗基地后直接重生新一轮，无任何玩家参与感，错失 Roguelike 的核心乐趣——"轮间成长选择"。

**实现方案**：
- 歼灭所有海盗基地后，显示 `EndlessUpgradePanel`（新建 `ui/EndlessUpgradePanel.lua`）
- 面板显示 30s 倒计时 + 3 张随机升级卡（从 15 张卡池中抽取，不重复）
- 玩家点击选择一张，效果永久生效（本局）；倒计时归零自动随机选1张
- 选完后执行原 `startEndlessNextRound()` 逻辑

**升级卡池（15 种）**：

| 类别 | 卡名 | 效果 |
|------|------|------|
| 战力 | 钛合金装甲 | 全舰耐久+25% |
| 战力 | 超频武装 | 全舰攻击力+20% |
| 战力 | 极速引擎 | 全舰移动速度+30% |
| 战力 | 快速装填 | 连击计时器+50% |
| 战力 | 自动修复 | 战斗中每5s全舰回血5% |
| 经济 | 丰厚矿脉 | 所有矿物产量+30% |
| 经济 | 精炼特效 | 精炼速率×1.5 |
| 经济 | 学分涌现 | 学分收入+40% |
| 经济 | 建造提速 | 所有建造时间-20% |

| 防御 | 护盾强化 | 基地护盾+800 |
| 防御 | 防御矩阵++ | 基地防御+25% |
| 特殊 | 虫洞通道 | WARP_GATE 解锁（如已有则 CD 减半）|
| 特殊 | 时间胶囊 | 回复20%已消耗的舰队耐久 |

### P2-2 战斗中途目标指令

**背景**：战斗全程自动进行，玩家只能在战前选阵型、战中用技能。缺少"战术干预"机会，战斗偏向观战体验。

**实现方案（低入侵方式）**：
- 在 `BattleScene.lua` 中新增 `focusTargetId_` 变量（nil = 自动选目标，非nil = 集中攻击）
- 战斗渲染层：敌舰上方显示血条，**点击敌舰**设置为集火目标（红色指示环 + 动态箭头）
- 当前集火目标在顶部显示小横幅："⚠ 集火目标：[敌舰名]"（3s 后淡出）
- 再次点击同一敌舰 → 取消集火，恢复自动；点击空白处 → 取消集火

**AI 逻辑改动**（最小化）：
- 各友方舰船攻击时，若 `focusTargetId_` 非 nil 且该目标存活 → 优先攻击集火目标
- 集火目标死亡后 `focusTargetId_` 自动清空

**视觉反馈**：
- 集火目标：红色 `nvgCircle` 虚线旋转环（每帧旋转 2°）+ 闪烁红色外晕
- 点击时：屏幕短暂闪一个红色扫射线（长度=屏幕宽度40%，持续0.3s）
- "连击加成"：持续集火同一敌舰消灭后，学分奖励+20%（新机制：专注击杀奖励）

### P3-2 EndGame 战绩雷达图

**背景**：EndGamePanel 展示得分/星级/统计，但缺乏"和自己/他人比较"的维度，结算页面信息量感不足。

**雷达图 6 个维度**：

|------|---------|---------|
| 殖民广度 | 殖民星球数 / 12 | 12星 = 100% |
| 战斗强度 | 总击杀数 / 200 | 200击杀 = 100% |
| 科技深度 | 解锁科技数 / 13 | 13项 = 100% |
| 经济规模 | 峰值精炼资源总量 / 5000 | 5000 = 100% |

- 图表下方：一句话评价（根据最高维度和最低维度生成，如"殖民强但经济薄弱"）

### P3-3 动态难度调整（DDA）

**背景**：normal 难度对熟练玩家过于轻松，hard 对新手过于困难，两者之间无平滑过渡。

**实现方案（非侵入式）**：
- 仅在 `normal` 和 `easy` 难度下激活 DDA（hard/custom 保持原值）
- 每 2 分钟采样一次玩家状态（资源量/殖民数/舰队战损比），计算"优势分"：
  ```
  advantageScore = colonized*10 + (metal/500) + (fleet_size-loss)*5
  ```
- 根据 advantageScore 微调 attackFactor：
  - 优势分 > 80：attackFactor 每轮 -5%（最大收紧到原值 × 0.7）
  - 优势分 < 30：attackFactor 每轮 +5%（最大放宽到原值 × 1.4）
  - 30~80：不调整
- DDA 调整幅度上限：基准值的 ±30%（保证调整透明，不破坏难度预期）
- 调整时不做任何提示（静默进行，保持沉浸感）
