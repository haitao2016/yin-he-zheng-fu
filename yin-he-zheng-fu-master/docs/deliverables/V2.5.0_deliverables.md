## 三、技术文档

### 3.1 代码架构

```
scripts/game/
├── galaxy/                    # 星图模块（新增）
│   ├── GalaxySceneGeneration.lua   # 星图生成
│   ├── GalaxySceneFleet.lua        # 编队管理
│   └── GalaxySceneSaveLoad.lua     # 存档管理
├── ui/                        # UI模块
│   ├── FleetPanel.lua              # 编队面板（主模块）
│   ├── FleetPanelNaming.lua        # 舰队命名（新增）
│   └── FleetPanelPresets.lua       # 编队预设（新增）
└── test/                      # 测试模块（新增）
    ├── smoke_test.lua              # 冒烟测试
    └── interface_checker.lua       # 接口检查
```

### 3.2 关键接口

| 模块 | 接口 | 参数 | 说明 |
|------|------|------|------|
| GameUI | `RefreshFleetPanel(fm, fid)` | fm:舰队管理器, fid:编队ID | 刷新编队面板 |
| GameUI | `ShowEndGame(gameType, stats, callback)` | gameType:游戏类型, stats:统计数据, callback:回调 | 显示结算面板 |
| ClientBattle | `OnFleetSiegeBase(fleetId, baseId)` | fleetId:舰队ID, baseId:基地ID | 舰队围攻海盗基地 |
| ClientGalaxy | `DoColonize(planet)` | planet:行星对象 | 殖民行星 |
| CampaignSystem | `CompleteLevel()` | 无 | 完成战役关卡 |
| LegacySystem | `AwardEndOfGame(stats)` | stats:统计数据表 | 发放文明遗产积分 |
